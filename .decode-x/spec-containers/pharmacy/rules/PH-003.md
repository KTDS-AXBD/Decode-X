# PH-003 — 처방 조제 atomic

**BL ID**: PH-003
**Domain**: Pharmacy (제약/약국)
**Detector**: AtomicTransaction
**Function**: `dispensePrescription()`

## Condition
처방 조제 요청 시

## Criteria
- `prescription.status = 'pending'`
- `drug.status != 'recalled'`
- `drug.stock_count > 0`

## Outcome
atomic: `stock_count` 차감 + `prescriptions.status='dispensed'` + `dispenses` INSERT

## Exception
- `E422-DRUG-RECALLED`: 리콜 약품 조제 불가
- `E422-OUT-OF-STOCK`: 재고 없음
- `E409-PRESCRIPTION`: 비정상 처방 상태
- `E404-PRESCRIPTION`, `E404-DRUG`: 미존재
