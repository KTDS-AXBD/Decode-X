# ED-002: 수강 학점 한도 검증

**Rule**: `MAX_CREDITS_PER_TERM - enrolledCredits ≥ requestedCredits` 충족 시 신청 가능

**구현 함수**: `checkCreditsLimit`
**Detector**: ThresholdCheck (F445 Path B — `creditsLimit < requestedCredits`, `limit` keyword)

## 트리거 조건
- **When**: 수강 신청 직전
- **If**: `student.status='active'` AND `creditsLimit ≥ requestedCredits`
- **Then**: `{canEnroll: true, remainingCreditsLimit}` 반환
- **Else**: `E404-ST` / `E409-ST` / `E422-LIMIT`

## 신뢰도
- Detector: 70% (Path B — `limit` keyword)
