# PT-001 — 펫호텔 정원 한도 검증

**함수**: `bookBoarding()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant fallback)

## 조건
펫호텔 예약 요청 시 `facility.booked_count >= capacity(또는 MAX_BOARDING_CAPACITY)` 검사.

## 기준
`MAX_BOARDING_CAPACITY = 30` (펫호텔 정원 기본 한도, 마리)

## 결과
- 정원 내: 예약 허용, boardings 생성 + boarding_facilities.booked_count 증가
- 정원 초과: `E422-BOARDING-CAPACITY-EXCEEDED` 예외

## 근거
반려동물 펫호텔 서비스의 시설별 수용 한도 관리 — 담당 직원 1인당 돌봄 가능 마리 수 제한.
