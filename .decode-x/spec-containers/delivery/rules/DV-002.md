# DV-002: 지역별 배송 한도 검증

**Rule**: 지역별 무게 한도 (RURAL ≤ 10kg, OVERSEAS ≤ 20kg, 기타 ≤ 30kg) 충족 검증

**구현 함수**: `checkRegionLimit`
**Detector**: ThresholdCheck (F445 Path B — `regionLimit < weight_kg` var-vs-var, `regionLimit`/`weight` 키워드 매칭)

## 트리거 조건
- **When**: 배송 시작 직전 (DV-003 호출 전 사전 검증)
- **If**: `delivery.status = 'PENDING'` AND `regionLimit ≥ weight_kg`
- **Then**: `{canShip: true, regionLimit}` 반환
- **Else**: `E404-DV` / `E409-ST` / `E422-RGN-LIMIT`

## 신뢰도
- Detector: 70% (ThresholdCheck Path B — var-vs-var with `limit` keyword)
