# Runbook — VD-002 시청자 일일 view 한도

## 알림 조건
`E422-DAILY-VIEW-LIMIT-EXCEEDED` 빈도 급증 시.

## 진단
1. `ad_contracts.view_used` vs `view_limit` 비교 (시청자별 일일 view)
2. 특정 시청자 등급(free/basic/premium/ad-free)별 한도 분포 점검
3. abuse pattern 의심 시 (bot view, 자동 새로고침) Cloudflare WAF 룰 적용 검토

## 해소
- 임시: 등급 업그레이드 또는 promo 제공 (광고 시청으로 한도 회복 옵션)
- 영구: `view_limit` 등급 재조정 또는 daily reset cron 정상 동작 확인 (KST 00:00 reset)
