# Runbook — EX-006 booth 환불 atomic

## 알림 조건
`E404-WITHDRAWN-ADMISSION` 또는 환불 잔액 불일치 의심 시.

## 진단
1. `admissions.status='withdrawn'` 조건 충족 확인
2. `refundAmount = boothCost × refundRate` 계산 검증
3. `db.transaction()` rollback 로그

## 해소

- 영구: refund rate 정책 (전액/부분/대체 일자 보상) 별 audit log + 전시주최기관 회신 hook
