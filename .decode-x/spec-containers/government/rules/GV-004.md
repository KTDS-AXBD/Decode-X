# GV-004 — 신청 상태 전환 (submitted → reviewing → approved → issued)

**BL ID**: GV-004
**Domain**: Government (공공/정부)
**Detector**: StatusTransition
**Function**: `transitionApplicationStatus()`

## Condition
허가 신청 상태 전환 시

## Criteria
허용 전환 매트릭스:
- `submitted → reviewing`
- `reviewing → approved`
- `approved → issued`
- `reviewing → rejected`
- `submitted → expired`

## Outcome
`permit_applications.status` UPDATE

## Exception
- `E404-APPLICATION`: 허가 신청 미존재
- `E409-APPLICATION`: 허용되지 않는 상태 전환
