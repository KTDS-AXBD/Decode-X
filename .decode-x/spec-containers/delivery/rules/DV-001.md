# DV-001: 배송 요청 무게 한도 검증

**Rule**: 0.1kg ≤ weight ≤ 30kg 충족 시 배송 요청 등록, 미충족 시 거절

**구현 함수**: `requestDelivery`
**Detector**: ThresholdCheck (F445 Path A — `weightKg < MIN_WEIGHT_KG` / `weightKg > MAX_WEIGHT_KG` UPPERCASE_CONSTANT)

## 트리거 조건
- **When**: 주문 시스템에서 배송 요청 호출
- **If**: `MIN_WEIGHT_KG ≤ weightKg ≤ MAX_WEIGHT_KG`
- **Then**: `deliveries` INSERT (status='PENDING') + `delivery_events` CREATED + expectedDeliveryAt 자동 산정 (OVERSEAS 7일 / RURAL 4일 / 기타 2일)
- **Else**: `E422-WT-MIN` (미달) 또는 `E422-WT-MAX` (초과)

## 신뢰도
- Detector: 70% (ThresholdCheck Path A — UPPERCASE_CONSTANT 비교)
