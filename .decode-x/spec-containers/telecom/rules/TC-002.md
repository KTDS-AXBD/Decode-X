# TC-002 — 데이터 사용량 한도 비교

**BL ID**: TC-002
**Domain**: Telecom (통신)
**Detector**: ThresholdCheck (Path B — var-vs-var, `dataQuotaLimit` keyword 매칭)
**Function**: `checkDataUsage()`

## Condition
데이터 사용량 기록 요청 시

## Criteria
- `usageBytes <= dataQuotaLimit` (누적 사용량 ≤ 요금제 데이터 할당량)
- `dataQuotaLimit`: `data_plans.data_quota_limit` 조회값 (F445 Path B `limit` keyword)

## Outcome
`data_usages` INSERT + 누적 사용량 기록

## Exception
- `E429-DATA-QUOTA`: 데이터 할당량 초과
