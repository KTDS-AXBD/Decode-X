# PR-001: 공과금 청구 한도 검증

**Detector**: ThresholdCheck (Path A — UPPERCASE constant)  
**Function**: `computeUtilityBill()`

## 규칙

공과금 청구 총액이 `MAX_UTILITY_BILL_AMOUNT`(500,000원)를 초과하면 처리를 거부한다.

## 조건

```
totalAmount >= MAX_UTILITY_BILL_AMOUNT → E422-UTILITY-BILL-EXCEEDED
```

## 근거

과도한 공과금 청구로 인한 임차인 보호 및 분쟁 예방.
