# Runbook — SH-001 선박 컨테이너 슬롯 동시 예약 정원 한도

## 알림 조건
`E422-VESSEL-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `vessels.active_bookings` vs `total_capacity` 비교 조회
2. 특정 선박/노선/시즌(peak season) 동시 예약 집중 여부 확인
3. `MAX_CONCURRENT_BOOKINGS_PER_VESSEL` 기본값(200 TEU) 적절성 검토 (peak vs 평시)

## 해소
- 임시: 화주에게 다른 선박 안내 또는 total_capacity 직접 조정 (피크 흡수)
- 영구: 선대 capacity 증설 또는 peak season 분산 (오프 시즌 인센티브)
