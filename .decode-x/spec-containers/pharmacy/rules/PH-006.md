# PH-006 — 약물 상호작용 atomic

**BL ID**: PH-006
**Domain**: Pharmacy (제약/약국)
**Detector**: AtomicTransaction
**Function**: `checkDrugInteraction()`

## Condition
신규 약물 처방 추가 전 상호작용 검증 시

## Criteria
- 처방 status != 'completed' AND != 'expired'
- severe 상호작용 발견 시 처방 차단

## Outcome
atomic: 상호작용 검증 + 대체약 추천 + `interaction_logs` INSERT + 처방 차단(blocked=1) 또는 허용(blocked=0)

## Exception
- `E409-PRESCRIPTION`: 완료/만료 처방에 상호작용 검증 불가
- `E404-PRESCRIPTION`, `E404-DRUG`: 미존재
