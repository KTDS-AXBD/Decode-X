# SP-001 — 경기장 좌석 한도 검증

**함수**: `bookVenueSeat()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant)

## 조건
좌석 예약 요청 시 `totalBooked >= MAX_VENUE_CAPACITY` 검사.

## 기준
`MAX_VENUE_CAPACITY = 50000` (경기장 최대 좌석 수)

## 결과
- 한도 내: 예약 허용, venue_seats 생성
- 한도 초과: `E422-VENUE-CAPACITY-EXCEEDED` 예외

## 근거
스포츠 시설 안전 규정 — 경기장 과잉 예약으로 인한 안전 사고 방지.
