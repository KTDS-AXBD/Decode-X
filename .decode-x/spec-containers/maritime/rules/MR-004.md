# MR-004 — 선적 상태 전환

**BL ID**: MR-004
**Domain**: Maritime (해운)
**Detector**: StatusTransition
**Function**: `transitionShipmentStatus()`

## Condition
선적 상태 전환 (booked → loaded → at_sea → arrived → delivered)

## Criteria
허용 매트릭스:
- booked → loaded
- loaded → at_sea
- at_sea → arrived
- arrived → delivered

## Outcome
shipments.status UPDATE + 타임스탬프 기록 (loaded_at / departed_at / arrived_at / delivered_at)

## Exception
- `E404-SHIPMENT`: 선적 미존재
- `E409-SHIPMENT`: 허용되지 않는 상태 전환 시도
