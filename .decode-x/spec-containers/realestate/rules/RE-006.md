# RE-006: 계약 취소 + 환불 atomic

**Rule**: active 검증 후 atomic하게 terminated + property listed 복구 (시작 14일 이전 시 deposit 100% 환불)

**구현 함수**: `cancelLeaseWithRefund`
**Detector**: AtomicTransaction (`db.transaction(() => { ... }); tx();`)

## 트리거 조건
- **When**: 양 당사자 취소 합의
- **If**: `lease.status='active'`
- **Then**: atomic transaction
  1. `UPDATE leases SET status='terminated', terminated_at=now`
  2. `UPDATE properties SET status='listed'` (재매물화)
- 환불 정책: 시작 14일 이전 → deposit 100% (refundAmount = deposit_krw), 미만 → 0원
- **Else**: `E404-LS` / `E409-LS`

## 무결성 보장
- 취소 + property 재매물화 atomic — 중복 임차/공실 누락 방지

## 신뢰도
- Detector: 85%
