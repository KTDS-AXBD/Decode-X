# BK-006 — KYC 본인확인 atomic

**BL ID**: BK-006
**Domain**: Banking (은행)
**Detector**: AtomicTransaction
**Function**: `verifyKyc()`

## Condition
KYC 본인확인 처리 시

## Criteria
- `account.status = 'pending_kyc'`
- 신원 서류 확인
- AML 위험도 평가

## Outcome
atomic: `kyc_records` INSERT(verified) + `aml_checks` INSERT(cleared) + `accounts.status='active'` UPDATE

## Exception
- `E404-ACCOUNT`: 계좌 미존재
- `E409-ACCOUNT`: pending_kyc 상태 아님
