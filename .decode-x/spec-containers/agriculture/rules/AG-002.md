# AG-002 — 살충제 살포 한도 비교

**BL ID**: AG-002
**Domain**: Agriculture (농업)
**Detector**: ThresholdCheck (Path B — var-vs-var, `pesticideQuotaLimit` keyword)
**Function**: `applyPesticide()`

## Condition
살충제 살포 요청 시

## Criteria
- `pesticideApplied <= pesticideQuotaLimit` (면적 카테고리별 최대 허용량)
- field.status != 'restricted'

## Outcome
살포 허용 + approved=true + pesticide_logs INSERT

## Exception
- `E422-PESTICIDE-QUOTA`: 살포 한도 초과
- `E409-FIELD`: 제한 구역 필드 (살포 금지)
- `E404-FIELD`: 필드 미존재
