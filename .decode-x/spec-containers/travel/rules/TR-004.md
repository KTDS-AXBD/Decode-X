# TR-004 — 여행 상태 전환 (pending → confirmed → checked_in → completed)

**BL ID**: TR-004
**Domain**: Travel (여행)
**Detector**: StatusTransition
**Function**: `transitionTripStatus()`

## Condition
여행 상태 변경 요청 시

## Criteria
허용 전환 매트릭스:
- `pending` → `confirmed`
- `confirmed` → `checked_in`
- `checked_in` → `completed`

## Outcome
`trips.status` UPDATE

## Exception
- `E404-TR`: 여행 미존재
- `E409-TR`: 허용되지 않은 상태 전환
