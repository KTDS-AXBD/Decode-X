# IN-004: 정책 상태 전환 (active ↔ suspended, → cancelled)

**Rule**: 허용된 상태 전환만 인정. 종료 상태에서는 전환 불가

**구현 함수**: `transitionPolicyStatus`
**Detector**: StatusTransition (`status` SELECT-then-UPDATE 패턴)

## 트리거 조건
- **When**: 사용자/관리자 일시 중지/재개/해약 요청
- **If**: 허용 매트릭스 충족
  - `active → suspended` (보험료 미납 등)
  - `active → cancelled` (해약)
  - `suspended → active` (재개)
  - `suspended → cancelled` (일시 중지 중 해약)
- **Then**: `UPDATE policies SET status=?, cancelled_at=?` (cancelled 시점만 cancelled_at)
- **Else**: `E409-TR`

## 상태 머신
```
active ↔ suspended
active → cancelled
suspended → cancelled
```

## 신뢰도
- Detector: 75% (StatusTransition)
