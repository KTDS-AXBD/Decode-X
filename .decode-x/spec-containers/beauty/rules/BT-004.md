# BT-004 — 예약 상태 전환

**함수**: `transitionAppointmentStatus()`
**Detector**: StatusTransition

## 조건
예약 상태 전환 요청 시 허용 매트릭스 검사.

## 기준
허용 전환:
- booked → confirmed
- confirmed → in_service
- in_service → completed
- completed → reviewed

## 결과
- 허용: appointments.status UPDATE
- 불허: `E409-APPOINTMENT` 예외

## 근거
미용 예약 라이프사이클 무결성 — 단계 건너뛰기 방지로 서비스 품질 추적.
