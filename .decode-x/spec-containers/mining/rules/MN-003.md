# MN-003 — 발파 작업 atomic

**함수**: `processBlastOperation()`
**Detector**: AtomicTransaction

## 조건
발파 작업 실행 시 planned 상태 확인 후 원자적 처리.

## 기준
`blast_operation.status = 'planned'`

## 결과
atomic: blast_records INSERT + safety_clearances INSERT + blast_operations.status 갱신 + ore_batches.status 갱신

## 근거
발파 작업은 안전 허가 + 기록 + 상태 갱신이 원자적으로 완결되어야 함. 부분 실행은 허용되지 않는다.
