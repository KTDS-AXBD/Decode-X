# DF-001 — 무기 보유 한도 검증

**함수**: `recordWeaponInventory()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant)

## 조건
무기 보유 기록 요청 시 `totalQuantity >= MAX_WEAPON_INVENTORY_LIMIT` 검사.

## 기준
`MAX_WEAPON_INVENTORY_LIMIT = 500` (단위부대 최대 무기 보유 수량)

## 결과
- 한도 내: 보유 허용, weapon_inventory 생성
- 한도 초과: `E422-INVENTORY-EXCEEDED` 예외

## 근거
군사 안전 규정 — 단위부대 무기 과잉 보유로 인한 안전 사고 방지.
