# CN-005 — 마일스톤 일괄 완료 처리

**BL ID**: CN-005
**Domain**: Construction (건설)
**Detector**: StatusTransition (batch 패턴, CC-005 20번째 재사용)
**Function**: `markMilestoneCompletion()`

## Condition
마일스톤 일괄 완료 배치 실행 시

## Criteria
- `milestones.completed = 0`
- `due_date <= dueDateCutoff`

## Outcome
`milestones.completed=1, status='completed'` 일괄 UPDATE

## Exception
대상 없으면 markedCount=0 (정상 반환)
