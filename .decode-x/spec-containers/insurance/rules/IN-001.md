# IN-001: 보험 가입 premium/age 한도 검증

**Rule**: premium ≥ 10,000 AND 18 ≤ age ≤ 80 충족 시 정책 발급, 미충족 시 거절

**구현 함수**: `issuePolicy`
**Detector**: ThresholdCheck (F445 Path A — `premiumKrw < MIN_PREMIUM_KRW` / `insuredAge < MIN_INSURED_AGE` / `insuredAge > MAX_INSURED_AGE` UPPERCASE)

## 트리거 조건
- **When**: 사용자 신규 보험 가입 신청
- **If**: `premiumKrw ≥ MIN_PREMIUM_KRW` AND `MIN_INSURED_AGE ≤ insuredAge ≤ MAX_INSURED_AGE`
- **Then**: `policies` INSERT (status='active', reserved_amount=0, expires_at = now + 1년)
- **Else**: `E422-PR` / `E422-AGE-MIN` / `E422-AGE-MAX`

## 신뢰도
- Detector: 70% (ThresholdCheck Path A — UPPERCASE_CONSTANT 비교)
