# DV-005: 배송 지연 자동 마킹

**Rule**: SHIPPED 상태 + 예상 배송일 + 3일 초과 시 status='DELAYED' 자동 마킹

**구현 함수**: `markDelayedDeliveries`
**Detector**: StatusTransition (`status='DELAYED'` 일괄 UPDATE 패턴, CC-005 동일 검증된 패턴)

## 트리거 조건
- **When**: 정기 batch (일/시간 단위, SYSTEM cron)
- **If**: `status='SHIPPED'` AND `expected_delivery_at < now - 3days`
- **Then**: `UPDATE deliveries SET status='DELAYED', delayed_since=asOfDate`, 마킹된 deliveryIds 반환
- **Else**: 정상 배송 marking 제외

## 모니터링
- 일별 DELAYED 신규 마킹 카운트 (대시보드)
- 지역별 지연율 (RURAL/OVERSEAS 운영 입력)

## 신뢰도
- Detector: 75% (StatusTransition — batch UPDATE 패턴, CC-005 입증된 동일 형태)
