# Runbook — AG-001: 단위 면적당 수확량 한도 검증

## 트리거
농부/수확SYSTEM → `POST /api/v1/agriculture/crops/:id/yield`

## 처리 흐름
1. 작물 status='mature' 또는 'harvested' 확인
2. 필드 면적(area_hectares)으로 단위 면적당 수확량 산출
3. `yieldPerHectare >= MAX_YIELD_PER_HECTARE` (10000 kg/ha) 차단
4. 검증 통과 시 yield_kg 기록 + status='harvested' 전환

## 모니터링
- E422-YIELD-EXCEEDED 발생률 (비정상 수확량 신고 비율)
- 수확량 임계값 근접 비율 (>9000 kg/ha)
