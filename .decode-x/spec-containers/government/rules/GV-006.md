# GV-006 — 문서 검증 Atomic (검증 + 인증 + 발급)

**BL ID**: GV-006
**Domain**: Government (공공/정부)
**Detector**: AtomicTransaction
**Function**: `validateDocument()`

## Condition
문서 검증 처리 요청 시

## Criteria
- `document.status = 'pending'`

## Outcome
atomic: `documents.status='validated'` + `documents.status='certified'` + `documents.status='issued'`

## Exception
- `E404-DOCUMENT`: 문서 미존재
- `E409-DOCUMENT`: 문서 상태 부적합 (pending이 아님)
