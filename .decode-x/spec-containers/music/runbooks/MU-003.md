# Runbook — MU-003 트랙 재생 atomic

## 알림 조건
`E404-SESSION` 또는 트랙 재생 atomic 실패 빈도 급증 시.

1. `playback_sessions.status = 'confirmed'` 조건 만족 여부
2. `db.transaction()` 실패 로그 분석 (DB lock, integrity constraint)
3. `track_plays` / `royalty_payouts` 삽입 실패 원인 추적
