# RE-001: 매물 등록 규모 한도 검증

**Rule**: 5 ≤ sizeM2 ≤ 10,000 충족 시 매물 등록

**구현 함수**: `listProperty`
**Detector**: ThresholdCheck (F445 Path A — UPPERCASE_CONSTANT)

## 트리거 조건
- **When**: 소유자 매물 등록
- **If**: `MIN_SIZE_M2 ≤ sizeM2 ≤ MAX_SIZE_M2`
- **Then**: `properties` INSERT (status='listed')
- **Else**: `E422-SZ-MIN` 또는 `E422-SZ-MAX`

## 신뢰도
- Detector: 70% (Path A)
