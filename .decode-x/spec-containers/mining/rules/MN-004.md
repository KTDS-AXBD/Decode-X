# MN-004 — 광석 상태 전환

**함수**: `transitionOreStatus()`
**Detector**: StatusTransition

## 조건
광석 배치 상태 전환 요청 시 허용 매트릭스 검사.

## 기준
허용 전환: extracted → graded → processed → shipped

## 결과
- 허용: ore_batches.status UPDATE + 타임스탬프 기록
- 불가: `E409-BATCH` 예외

## 근거
광석 처리 단계 순서 보장 — 미선별(graded) 미처리(processed) 광석의 출하(shipped) 방지.
