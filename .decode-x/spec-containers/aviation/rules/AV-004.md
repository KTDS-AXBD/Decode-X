# AV-004 — 비행 상태 전환

**함수**: `transitionFlightStatus()`
**Detector**: StatusTransition

## 조건
비행 상태 전환 요청 시 허용 매트릭스 검사.

## 상태 전이 허용 매트릭스
```
scheduled  → boarding
boarding   → departed
departed   → in_flight
in_flight  → landed
landed     → completed
```

## 결과
- 허용 전환: `flights.status` UPDATE + 타임스탬프 기록
- 비허용 전환: `E409-FLIGHT` 예외

## 근거
항공 운항 단계 순서 보장 — 역행 및 건너뜀 금지.
