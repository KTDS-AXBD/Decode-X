# SB-002: 자동결제 한도 검증

**Rule**: 자동 결제 직전 `auto_charge_limit ≥ price_krw` 검증

**구현 함수**: `checkAutoChargeLimit`
**Detector**: ThresholdCheck (F445 Path B — `auto_charge_limit < price_krw` var-vs-var, `limit` keyword 매칭)

## 트리거 조건
- **When**: SB-003 자동 갱신 직전 사전 검증
- **If**: `sub.status='active'` AND `auto_charge_limit ≥ price_krw`
- **Then**: `{canCharge: true, remainingHeadroom}` 반환
- **Else**: `E404-SB` / `E409-ST` / `E422-LIMIT`

## 신뢰도
- Detector: 70% (ThresholdCheck Path B — `limit` keyword)
