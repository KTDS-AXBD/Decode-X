# Runbook — PC-006 청취자 환불 atomic

## 알림 조건
`E404-REMOVED-DISTRIBUTION` 또는 환불 잔액 불일치 의심 시.

## 진단
1. `episode_distributions.status='removed'` 조건 충족 확인
2. `refundAmount = subscriptionCost × refundRate` 계산 검증 (소수점 2자리 반올림)
3. `db.transaction()` rollback 로그

- 영구: refund rate 정책 (전액/부분/명예회복) 별 audit log + 결제수단 회신 (카드사 chargeback) audit
