# Runbook — P-004: 수령 개시 시기 검증

**Rule**: 연령 ≥ 55세 AND 가입 기간 ≥ 5년이면 수령 개시 허용

## 트리거
- `POST /api/v1/pension/{accountId}/receipt/initiate`

## 전제조건
- `applicantAge`: 신청자 현재 연령
- `subscriptionYears`: 가입 후 경과 연수 (개월 기준 소수점 가능)
- `MIN_RECEIPT_AGE = 55`, `MIN_SUBSCRIPTION_YEARS = 5`

## 처리 흐름
1. `applicantAge < MIN_RECEIPT_AGE` → `E422-AGE_NOT_MET` 반환
2. `subscriptionYears < MIN_SUBSCRIPTION_YEARS` → `E422-SUBSCRIPTION_TOO_SHORT` 반환
3. 통과 시 → `pension_accounts.status = 'RECEIVING'` 갱신 + 수령 스케줄 생성 + `ReceiptInitiated` 이벤트
