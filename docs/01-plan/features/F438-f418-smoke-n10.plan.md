## 검증 시나리오

### 단위 (각 inference 호출)
- Request: chunks 1~2건 → `POST /policies/infer` (orgId=`org-smoke-f438-{seq}`)
- Response: 8±2 candidates, HTTP 200
- D1 검증: `SELECT id, exception, outcome FROM policies WHERE organization_id=? ORDER BY created_at` → exception NOT NULL 비율 측정

### 통계 (n=10 누적)
- 메인 정책 exception 채움률 (분모: 메인 정책 수, 분자: exception NOT NULL)
- 별도 EX 정책 비율 (전체 candidates 중 EX-* prefix 비율)
- 종합 자연 채움률 (분모: 전체 candidates, 분자: exception NOT NULL)
- 평균 ≥ 50% + 표준편차 ≤ 15%pp 검증

### 회귀
- production svc-policy `/health` HTTP 200 유지 (smoke 호출이 production 영향 없음 검증)
- D1 db-policy 비smoke row 변경 없음 (orgId 격리)
