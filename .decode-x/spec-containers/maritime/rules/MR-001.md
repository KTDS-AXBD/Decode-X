# MR-001 — 화물 적재 한도 검증

**BL ID**: MR-001
**Domain**: Maritime (해운)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `loadCargo()`

## Condition
화물 선적 시

## Criteria
- `cargoTons < MAX_CARGO_CAPACITY_TONS` (최대 화물 적재 한도 200,000 tons)
- 선박 status='available' 또는 'loading' 확인

## Outcome
선적 허용 + shipments INSERT + vessels.current_load_tons 갱신

## Exception
- `E422-CARGO-LIMIT`: 화물 톤수 한도 초과 (200,000 tons)
- `E422-VESSEL-CAPACITY`: 선박 개별 적재 용량 초과
- `E409-VESSEL`: 비정상 선박 상태
- `E404-VESSEL`: 선박 미존재
