# PH-004 — 처방 상태 전환

**BL ID**: PH-004
**Domain**: Pharmacy (제약/약국)
**Detector**: StatusTransition
**Function**: `transitionPrescriptionStatus()`

## Condition
처방 상태 변경 요청 시

## Criteria
허용 전환 매트릭스:
- issued → pending (용량 검증 통과)
- issued → expired (유효기간 만료)
- pending → dispensed (조제 완료)
- pending → expired (유효기간 만료)
- dispensed → completed (복약 완료)
- dispensed → expired (유효기간 만료)
- completed → expired (최종 만료)

## Outcome
`prescriptions.status` UPDATE + 해당 타임스탬프(dispensed_at / completed_at / expired_at) 기록

## Exception
- `E409-PRESCRIPTION`: 허용되지 않는 상태 전환
- `E404-PRESCRIPTION`: 처방 미존재
