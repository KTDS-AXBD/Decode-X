# CN-002 — 보유금 비율 한도 비교

**BL ID**: CN-002
**Domain**: Construction (건설)
**Detector**: ThresholdCheck (Path B — var-vs-var, `retentionRateLimit` keyword)
**Function**: `computePaymentRetention()`

## Condition
대금 지급 보유금 계산 시

## Criteria
- `retentionRate <= retentionRateLimit` (계약 규모별 최대 보유금 비율: small=10%, medium=7%, large=5%)
- 프로젝트 status='in_progress' or 'completed' 확인

## Outcome
보유금 허용 + payment_retentions INSERT

## Exception
- `E422-RETENTION-EXCEEDED`: 보유금 비율 한도 초과
- `E409-PROJECT`: 비정상 프로젝트 상태
- `E404-PROJECT`: 프로젝트 미존재
