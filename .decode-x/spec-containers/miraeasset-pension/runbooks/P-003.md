# Runbook — P-003: 중도인출 사유 검증

**Rule**: 법정 허용 사유 해당 시 중도인출 요청 상태 '심사 중' 전환

## 트리거
- `POST /api/v1/pension/{accountId}/withdraw/early`

## 전제조건
- `withdrawalReason`: 인출 사유 코드
  - 허용 코드: `HOUSING_PURCHASE`, `HOUSING_LEASE`, `MEDICAL`, `DISASTER`, `LONG_TERM_CARE`
- `amount`: 인출 요청 금액

## 처리 흐름
1. `withdrawalReason` 허용 목록 외 → `E403-INVALID_REASON` 반환
2. `amount <= 0` → `E422-INVALID_AMOUNT` 반환
3. 통과 시 → `pension_withdrawals` INSERT (status='UNDER_REVIEW') + `EarlyWithdrawalRequested` 이벤트
