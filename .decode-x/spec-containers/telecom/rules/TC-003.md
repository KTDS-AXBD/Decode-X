# TC-003 — 플랜 업그레이드 atomic 트랜잭션

**BL ID**: TC-003
**Domain**: Telecom (통신)
**Detector**: AtomicTransaction
**Function**: `upgradePlan()`

## Condition
플랜 업그레이드 요청 시

## Criteria
- `subscription.status = 'active'` (활성 회선만 플랜 변경 가능)

## Outcome
atomic: `subscriptions.plan_id` UPDATE + `billing_cycles` INSERT (신규 청구) + `subscriptions.activated_at` 갱신

## Exception
- `E404-SUBSCRIPTION`: 회선 미존재
- `E409-SUBSCRIPTION`: 비활성 회선 플랜 변경 시도
- `E404-PLAN`: 대상 요금제 미존재
