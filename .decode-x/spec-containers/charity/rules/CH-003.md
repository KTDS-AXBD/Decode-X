# CH-003 — 기금 집행 atomic

**함수**: `disburseFund()`
**Detector**: AtomicTransaction

## 조건
기금 집행 요청 시 승인된 보조금 존재 필수.

## 기준
`grants.status = 'approved'`

## 결과
- 정상: `fund_disbursements` INSERT + `grants` status UPDATE + `disbursement_receipts` INSERT + `fund_disbursements.status` UPDATE (atomic)
- 보조금 없음: `E404-GRANT` 예외

## 근거
기금 집행은 원자적으로 처리되어야 함 — 중간 실패 시 전체 롤백으로 이중 집행 방지.
