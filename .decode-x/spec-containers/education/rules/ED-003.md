# ED-003: 수강 신청 atomic transaction

**Rule**: course capacity 검증 후 atomic하게 enrollment INSERT + course enrolled_count++

**구현 함수**: `enrollCourse`
**Detector**: AtomicTransaction (`db.transaction(() => { ... }); tx();`)

## 트리거 조건
- **When**: 학생 수강 신청
- **If**: `course.status='open'` AND `enrolled_count < capacity`
- **Then**: atomic transaction
  1. `INSERT INTO enrollments (status='enrolled')`
  2. `UPDATE courses SET enrolled_count = enrolled_count + 1`
- **Else**: `E404-CO` / `E409-CO` / `E422-FULL`

## 무결성 보장
- 두 변경 atomic — capacity over-enrollment 방지

## 신뢰도
- Detector: 85% (AtomicTransaction)
