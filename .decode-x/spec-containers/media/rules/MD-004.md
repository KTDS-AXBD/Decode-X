# MD-004 — 콘텐츠 상태 전환

**BL ID**: MD-004
**Domain**: Media (미디어)
**Detector**: StatusTransition
**Function**: `transitionContentStatus()`

## Condition
콘텐츠 상태 전환 시 (draft → reviewing → published → archived/expired)

## Criteria
허용 전환 매트릭스:
- draft → reviewing
- reviewing → published
- reviewing → archived
- published → archived
- published → expired
- archived → expired

## Outcome
`contents.status` UPDATE + published_at / archived_at 타임스탬프 기록

## Exception
- `E404-CONTENT`: 콘텐츠 미존재
- `E409-CONTENT`: 허용되지 않는 상태 전환
