# RT-006 — 반품 + 환불 Atomic (반품 접수 + 재고 복구 + 환불 처리)

**BL ID**: RT-006
**Domain**: Retail (소매)
**Detector**: AtomicTransaction
**Function**: `processReturnRefund()`

## Condition
반품 + 환불 트랜잭션 처리 시

## Criteria
`order.status IN ('delivered', 'completed')`

## Outcome
atomic: `orders.status='returned'` + `return_log` INSERT (refund_amount = total_amount)

## Exception
- `E404-ORD`: 주문 미존재
- `E409-ORD`: 반품 불가 상태 (requires delivered or completed)
