# Runbook — CC-005: 연체 자동 마킹

## 트리거
정기 batch (일/시간 단위, SYSTEM cron)

## 처리 흐름
1. cutoff = `now - 30days`
2. 후보 조회: `status='active'` AND `used_amount > 0` AND `last_payment_at < cutoff`
3. 각 cardId에 대해:
   - `UPDATE credit_cards SET status='delinquent', delinquent_since=asOfDate`
4. `{markedCount, delinquentCardIds}` 반환

## 알림
- delinquent 마킹 시 사용자 통보 + 신용평가 reporting
- markedCount > 일평균 × 2 시 dashboard alert (이상 신호)

## 모니터링
- 일별 delinquent 신규 마킹 카운트
- delinquent_since 분포 (만성 연체자 비율)
