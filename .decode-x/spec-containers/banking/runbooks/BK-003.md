# Runbook — BK-003: 계좌 이체 atomic

## 트리거
고객/이체SYSTEM → `POST /api/v1/banking/accounts/transfer`

## 처리 흐름
1. fromAccount status='active' + toAccount status='active' 확인
2. 잔액 충분 확인 (amount + fee)
3. atomic: fromAccount.balance -= (amount+fee) + toAccount.balance += amount + transactions INSERT

## 모니터링
- 이체 실패율 (E409-ACCOUNT / E422-INSUFFICIENT-FUNDS)
- atomic 트랜잭션 성공율
