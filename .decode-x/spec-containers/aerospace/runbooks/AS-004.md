# Runbook — AS-004 미션 상태 전환 매트릭스

## 알림 조건
`E409-MISSION` 빈도 급증 시.

## 진단
1. `missions` 상태 전환 시도 로그 확인 (currentStatus → newStatus 시도 매트릭스)
2. 허용 전환 외 시도가 어떤 caller(scheduler/operator/external)에서 발생하는지 확인
3. 외부 발사 시스템과 우리 시스템의 status 동기화 timing 점검

## 해소
- 즉시: 잘못된 전환 시도 caller 차단 또는 status sync 재실행
- 영구: 외부 시스템 status webhook subscribe + idempotent reconciliation 도입
