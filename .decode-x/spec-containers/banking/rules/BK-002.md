# BK-002 — 송금 수수료 한도 비교

**BL ID**: BK-002
**Domain**: Banking (은행)
**Detector**: ThresholdCheck (Path B — var-vs-var, `limit` keyword)
**Function**: `computeTransferFee()`

## Condition
송금 처리 시 수수료 계산

## Criteria
- `feeAmount <= transferFeeLimit` (var-vs-var, `transferFeeLimit` — `limit` keyword 매칭)
- 계좌 유형별 수수료율 적용 (checking 0.1% / savings 0.05% / business 0.15%)
- 최대 수수료 한도: 50,000원

## Outcome
수수료 계산 완료 + 송금 허용

## Exception
- `E422-TRANSFER-FEE`: 수수료 한도 초과 (5만원)
