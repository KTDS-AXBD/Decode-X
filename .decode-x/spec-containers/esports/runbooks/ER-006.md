# Runbook — ER-006 prize 회수 atomic

## 알림 조건
`E404-FORFEITED-MATCH` 또는 회수 잔액 불일치 (`prize_clawback_records.status='clawed_back'`인데 `prize_clawbacks` row 누락) 의심 시.

## 진단
1. `matches.status='forfeited'` 조건 충족 확인 (archived 매치는 회수 대상 아님)
2. `clawbackAmount = prizeCost × clawbackRate` 계산 검증 (소수점 2자리 반올림, 부분 회수 정책 확인)
3. `db.transaction()` rollback 로그 — atomic 실패 시 records/clawbacks 양쪽 모두 미생성

## 해소
- 임시: 부분 실패 row 수동 정리 후 재시도
- 영구: clawback rate 정책 (전액/부분/match fixing 페널티) 별 audit log 추가 + 팀 통보 워크플로 + 게임사 결정 hook
