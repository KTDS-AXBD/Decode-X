# Runbook — AR-006 commission 환불 atomic

## 알림 조건
`E404-WITHDRAWN-ARTWORK` 또는 환불 잔액 불일치 의심 시.

## 진단
1. `artworks.status='withdrawn'` 조건 충족 확인
2. `refundAmount = commissionCost × refundRate` 계산 검증
3. `db.transaction()` rollback 로그

## 해소
- 임시: 부분 실패 row 수동 정리 후 재시도
- 영구: refund rate 정책 (전액/부분/감정평가) 별 audit log + 갤러리 회신 hook
