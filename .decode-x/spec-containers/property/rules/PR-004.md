# PR-004: 임대 상태 전환

**Detector**: StatusTransition  
**Function**: `transitionLeaseStatus()`

## 상태 머신

```
pending → active → renewed → terminated → archived
               ↘ terminated
```

## 허용 전환

| From | To |
|------|----|
| pending | active |
| active | renewed |
| active | terminated |
| renewed | terminated |
| terminated | archived |

## 근거

임대 계약 lifecycle 무결성 보장.
