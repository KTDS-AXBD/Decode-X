# Runbook — MD-003: 콘텐츠 라이선싱 atomic

## 트리거
구독자/라이선스SYSTEM → `POST /api/v1/media/contents/:id/license`

## 처리 흐름
1. content.status='published' 확인
2. license_count > 0 확인 (잔여 라이선스)
3. atomic: license_count 차감 + licenses INSERT
4. licenseId 반환

## 모니터링
- E422-LICENSE-EXHAUSTED 발생률 (라이선스 소진 콘텐츠 비율)
- 라이선스 발급 속도 (동시 요청 경합 감지)
