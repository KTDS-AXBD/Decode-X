# Runbook — ED-005: 학기 만료 미수료 자동 마킹

## 트리거
학기 종료 직후 batch (SYSTEM cron)

## 처리 흐름
1. term.term_end < now 확인
2. enrolled 상태인 모든 enrollment 조회
3. 각 enrollment에 대해:
   - `UPDATE enrollments SET status='failed', completed_at=now`
4. `{markedCount, failedEnrollmentIds}` 반환

## 알림
- failed 마킹 시 학생 SMS/email + 학적 시스템 통지
- 과목별 실패율 차원 분석 dashboard

## 모니터링
- 학기별 failed 마킹 카운트
- 과목별 미수료율 (커리큘럼 입력)
