# Runbook — DV-005: 배송 지연 자동 마킹

## 트리거
정기 batch (일/시간 단위, SYSTEM cron)

## 처리 흐름
1. cutoff = `now - 3days`
2. 후보 조회: `status='SHIPPED'` AND `expected_delivery_at < cutoff`
3. 각 deliveryId에 대해:
   - `UPDATE deliveries SET status='DELAYED', delayed_since=asOfDate`
4. `{markedCount, delayedDeliveryIds}` 반환

## 알림
- DELAYED 마킹 시 사용자 통보 + 배송사 추적 요청
- markedCount > 일평균 × 2 시 dashboard alert (이상 신호 — 기상/물류 사고)

## 모니터링
- 일별 DELAYED 신규 마킹 카운트
- 지역별 지연율 (RURAL/OVERSEAS 운영 입력)
