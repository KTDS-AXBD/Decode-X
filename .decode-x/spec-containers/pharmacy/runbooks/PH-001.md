# Runbook — PH-001: 일일 최대 용량 한도 검증

## 트리거
약사/조제SYSTEM → `POST /api/v1/pharmacy/prescriptions/:id/validate`

## 처리 흐름
1. 처방 status='issued' 또는 'pending' 확인
2. `dosageAmount >= MAX_DAILY_DOSE` (4000mg) 차단
3. 검증 통과 시 status='pending' 전환

## 모니터링
- E422-DOSAGE-EXCEEDED 발생률 (과용량 처방 비율)
- 용량 임계값 근접 처방 비율 (>3500mg)
