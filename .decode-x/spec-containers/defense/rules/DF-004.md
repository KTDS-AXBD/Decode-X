# DF-004 — 임무 상태 전환

**함수**: `transitionMissionStatus()`
**Detector**: StatusTransition

## 조건
임무 상태 전환 요청 시 허용 매트릭스 검사.

## 기준
허용 전환: planned → briefed → executing → completed → debriefed

## 결과
- 허용 전환: `missions.status` UPDATE + 타임스탬프 기록
- 불허 전환: `E409-MISSION` 예외

## 근거
임무 수행 단계별 순서를 강제함으로써 작전 절차 준수 및 기록 무결성 보장.
