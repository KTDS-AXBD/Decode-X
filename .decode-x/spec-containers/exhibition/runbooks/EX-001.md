# Runbook — EX-001 venue 동시 active admission 한도

## 알림 조건
`E422-VENUE-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `venues.active_admissions` vs `total_capacity` 비교 조회
2. 특정 전시장/대형 박람회(IFA/CES/MWC급) 동시 입장 집중 여부 확인
3. `MAX_CONCURRENT_ACTIVE_ADMISSIONS_PER_VENUE` 기본값(10000) 적절성 검토 (소방법 안전 수용 인원 기준)

## 해소
- 임시: venue에 closed 후보 admission 정리 권유 또는 total_capacity 직접 조정
- 영구: 시간대 분산 입장권/일자 분산 + 전시 회차 분할 + 소방법 안전 수용 인원 재산정
