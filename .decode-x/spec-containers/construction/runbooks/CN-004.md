# Runbook — CN-004: 프로젝트 상태 전환

## 트리거
프로젝트관리자/프로젝트관리SYSTEM → `PATCH /api/v1/construction/projects/:id/status`

## 처리 흐름
1. 허용 전환 매트릭스 검증 (bidding→awarded→in_progress→completed→closed)
2. projects.status UPDATE + 타임스탬프 기록

## 모니터링
- E409-PROJECT 발생률 (잘못된 상태 전환 시도)
- 단계별 평균 소요 기간 (bidding→awarded, awarded→in_progress 등)
