# MN-002 — 로열티 한도 비교

**함수**: `computeRoyalty()`
**Detector**: ThresholdCheck (Path B, var-vs-var)

## 조건
로열티 계산 시 `royaltyAmount > royaltyTierLimit` 검사.

## 기준
`royaltyTierLimit = royalty_tiers.max_royalty_limit` (단계별 분기 최대 로열티)

## 결과
- 한도 내: 로열티 승인 + royalty_records INSERT
- 한도 초과: `E422-ROYALTY-EXCEEDED` 예외

## 근거
광업권 계약 — 단계별 로열티 상한 준수 의무.
