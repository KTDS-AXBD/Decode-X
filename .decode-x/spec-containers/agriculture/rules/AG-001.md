# AG-001 — 단위 면적당 수확량 한도 검증

**BL ID**: AG-001
**Domain**: Agriculture (농업)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `recordCropYield()`

## Condition
작물 수확량 기록 시

## Criteria
- `yieldPerHectare < MAX_YIELD_PER_HECTARE` (단위 면적당 최대 10000 kg/ha)
- 작물 status='mature' or 'harvested' 확인

## Outcome
수확량 허용 + status='harvested' 전환

## Exception
- `E422-YIELD-EXCEEDED`: 단위 면적당 최대 수확량 초과 (10000 kg/ha)
- `E409-CROP`: 비정상 작물 상태
- `E404-CROP`: 작물 미존재
