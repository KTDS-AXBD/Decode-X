# Runbook — SH-002 화주 운임 수수료 한도 초과

## 알림 조건
`E422-FREIGHT-PAYMENT-LIMIT-EXCEEDED` 빈도 급증 시.

## 진단
1. `freight_contracts.fee_used` vs `fee_limit` 비교 조회
2. 화주별 누적 운임 수수료 사용 패턴 분석 (월별 / 분기별)
3. 등급(tier_code) 별 freightPaymentLimit 적정성 검토 (standard/priority/premium/reefer)

## 해소
- 임시: 화주에게 등급 상향 안내 또는 fee_limit 1회성 인상
- 영구: 등급별 한도 재산정 + 자동 알림(80% 도달 시 사전 경고)
