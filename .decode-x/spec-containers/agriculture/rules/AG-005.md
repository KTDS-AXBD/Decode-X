# AG-005 — 등급 일괄 갱신 (batch grading)

**BL ID**: AG-005
**Domain**: Agriculture (농업)
**Detector**: StatusTransition (batch 패턴 — CC-005 batch 19번째 재사용)
**Function**: `markBatchGrading()`

## Condition
등급 대상 수확물 일괄 처리 (배치)

## Criteria
- harvests.graded = 0 (미등급)
- harvested_at <= gradingCutoffDate

## Outcome
harvests.status='graded' + graded=1 일괄 UPDATE

## Exception
대상 없으면 markedCount=0 (정상 종료)
