# FT-001 — 클래스 정원 한도 검증

**함수**: `bookClassSlot()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant fallback)

## 조건
클래스 예약 요청 시 `cls.booked_count >= capacity(또는 MAX_CLASS_CAPACITY)` 검사.

## 기준
`MAX_CLASS_CAPACITY = 25` (클래스 정원 기본 한도, 인원)

## 결과
- 정원 내: 예약 허용, class_bookings 생성 + fitness_classes.booked_count 증가
- 정원 초과: `E422-CLASS-CAPACITY-EXCEEDED` 예외

## 근거
피트니스 센터의 클래스별 정원 관리 — 강사 1인당 지도 가능 인원 제한.
