# CN-003 — 변경 청구 원자 트랜잭션

**BL ID**: CN-003
**Domain**: Construction (건설)
**Detector**: AtomicTransaction
**Function**: `processChangeOrder()`

## Condition
변경 청구 처리 시

## Criteria
- `project.status = 'in_progress'`

## Outcome
atomic: `change_orders` INSERT + `projects.contract_value` 갱신 + `change_order_approvals` INSERT

## Exception
- `E404-PROJECT`: 프로젝트 미존재
- `E409-PROJECT`: 비정상 프로젝트 상태 (in_progress 아님)
