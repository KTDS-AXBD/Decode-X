# DF-003 — 임무 파견 atomic

**함수**: `dispatchMission()`
**Detector**: AtomicTransaction

## 조건
임무 파견 요청 시 planned 상태 확인 후 원자적 처리.

## 기준
`mission.status = 'planned'`

## 결과
atomic: missions INSERT + mission_assignments INSERT + mission_equipment INSERT + mission_communications INSERT

## 근거
임무 파견은 인원 배정 + 장비 할당 + 통신 활성화가 원자적으로 완결되어야 함. 부분 실행은 허용되지 않는다.
