# Runbook — GA-006 wager 환불 atomic

## 알림 조건
`E404-VOIDED-BET` 또는 환불 잔액 불일치 의심 시.

## 진단
1. `bets.status='voided'` 조건 충족 확인
2. `refundAmount = wagerAmount × refundRate` 계산 검증
3. `db.transaction()` rollback 로그

## 해소
- 임시: 부분 실패 row 수동 정리 후 재시도
- 영구: refund rate 정책 (전액/부분/책임도박) 별 audit log + 규제기관 (KGC/MGA) 회신 hook
