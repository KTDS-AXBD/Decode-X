# FT-004 — 운동 진행 상태 전환

**함수**: `transitionProgressStatus()`
**Detector**: StatusTransition

## 조건
회원 운동 프로그램 진행 상태 전환 요청.

## 기준
허용 전환 매트릭스:
- `initial → in_progress`
- `in_progress → assessment`
- `assessment → completed`

## 결과
- 허용 전환: member_progress.status UPDATE + 타임스탬프 기록
- 불허 전환: `E409-PROGRESS` 예외

## 근거
피트니스 트레이닝 프로그램의 단계적 진행 상태 관리 — 역행 전환 방지.
