# Runbook — SM-002 크리에이터 일일 monetization 한도

## 알림 조건
`E422-DAILY-MONETIZATION-LIMIT-EXCEEDED` 빈도 급증 시.

## 진단
1. `monetization_contracts.monetization_used` vs `monetization_limit` 비교 (크리에이터별 일일 monetization)
2. 특정 크리에이터 등급(free/basic/partner/premium)별 한도 분포 점검
3. abuse pattern 의심 시 (fake engagement, bot view) Cloudflare WAF 룰 + 광고 fraud 분석 적용

## 해소
- 임시: 등급 업그레이드 (partner → premium 승격 검토) 또는 promo 제공
- 영구: `monetization_limit` 등급 재조정 또는 daily reset cron 정상 동작 확인 (KST 00:00 reset) + payout 합산 로직 audit
