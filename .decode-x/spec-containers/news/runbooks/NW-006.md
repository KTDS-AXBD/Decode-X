# Runbook — NW-006 구독료 환불 atomic

## 알림 조건
`E404-RETRACTED-SYNDICATION` 또는 환불 잔액 불일치 (`subscription_refund_records.status='refunded'`인데 `subscription_refunds` row 누락) 의심 시.

## 진단
1. `article_syndications.status='retracted'` 조건 충족 확인 (archived 신디케이션은 환불 대상 아님)
2. `refundAmount = subscriptionCost × refundRate` 계산 검증 (소수점 2자리 반올림, 일할 계산 정책 확인)
3. `db.transaction()` rollback 로그 — atomic 실패 시 records/refunds 양쪽 모두 미생성

## 해소
- 임시: 부분 실패 row 수동 정리 후 재시도
- 영구: refund rate 정책 (전액/부분/명예회복금) 별 audit log 추가 + 정정 보도 사유 enum 도입 + 결제수단 회신 (카드사 chargeback) audit
