# PT-003 — 그루밍 예약 확정 Atomic

**함수**: `processGrooming()`
**Detector**: AtomicTransaction

## 조건
그루밍 예약 확정 요청 시 `groomings.status = 'booked'` 확인.

## 기준
예약 내역 존재 + 상태 'booked' 확인

## 결과
atomic: groomings UPDATE + grooming_payments INSERT + grooming_owner_matches INSERT + groomers UPDATE

## 예외
- `E404-GROOMING`: 예약 미존재

## 근거
그루밍 서비스 예약 확정 시 결제 + 보호자-그루머 매칭 + 그루머 상태 업데이트 일관성 보장.
