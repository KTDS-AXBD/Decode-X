# AG-003 — 수확 처리 atomic

**BL ID**: AG-003
**Domain**: Agriculture (농업)
**Detector**: AtomicTransaction
**Function**: `processHarvest()`

## Condition
수확 처리 요청 시

## Criteria
- crop.status = 'mature'
- field.status != 'restricted'

## Outcome
atomic: harvests INSERT + gradings INSERT + crops.status='harvested'

## Exception
- `E404-CROP`: 작물 미존재
- `E409-CROP`: 비수확 가능 상태 (mature 아님)
- `E404-FIELD`: 필드 미존재
- `E422-FIELD-RESTRICTED`: 제한 구역 필드에서 수확 불가
