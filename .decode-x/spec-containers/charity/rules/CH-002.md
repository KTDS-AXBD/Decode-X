# CH-002 — 보조금 한도 비교

**함수**: `applyGrant()`
**Detector**: ThresholdCheck (Path B, var-vs-var, grantTierLimit keyword)

## 조건
보조금 신청 시 `requestedAmount > grantTierLimit` 검사.

## 기준
`grantTierLimit = grant_tiers.grantTierLimit` (단계별 보조금 한도)

## 결과
- 한도 내: 보조금 승인
- 한도 초과: `E422-GRANT-TIER-EXCEEDED` 예외

## 근거
보조금 단계별 배분 규정 — 소규모(seed) / 일반(standard) / 대형(major) / 대표(lead) 차등 배분.
