# HO-003 — 체크인 Atomic Transaction

**BL ID**: HO-003
**Domain**: Hospitality (숙박)
**Detector**: AtomicTransaction
**Function**: `processCheckIn()`

## Condition
체크인 처리 시 (booking.status='confirmed')

## Criteria
- `booking.status='confirmed'`
- 배정 가능 객실 존재 (roomId 유효)

## Outcome
Atomic: `bookings.status='checked_in'` UPDATE + `rooms.status='occupied'` UPDATE

## Exception
- `E404-BK`: 예약 미존재
- `E409-BK`: 상태 불일치 (confirmed 아님)
- `E409-RM`: 객실 이미 점유
