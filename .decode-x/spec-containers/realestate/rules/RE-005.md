# RE-005: 계약 만료 자동 처리

**Rule**: active + end_at 도달 시 status='expiring' 자동 마킹 (갱신 안내 트리거)

**구현 함수**: `markExpiringLeases`
**Detector**: StatusTransition (`status='expiring'` 일괄 UPDATE 패턴, CC/DV/SB/IN/HC/ED-005 동일 형태 7번째)

## 트리거 조건
- **When**: 정기 batch (시간 단위, SYSTEM cron)
- **If**: `status='active'` AND `end_at < now`
- **Then**: `UPDATE leases SET status='expiring'`, 마킹된 leaseIds 반환
- **Else**: end_at 미도달 시 skip

## 모니터링
- 일별 expiring 마킹 카운트
- 갱신률 (expiring → renewed / expiring → terminated)

## 신뢰도
- Detector: 75% (StatusTransition — batch UPDATE, 7번째 동일 형태)
