# MD-003 — 콘텐츠 라이선싱 atomic

**BL ID**: MD-003
**Domain**: Media (미디어)
**Detector**: AtomicTransaction
**Function**: `processLicensing()`

## Condition
콘텐츠 라이선스 요청 시

## Criteria
- `content.status = 'published'`
- `license_count > 0` (잔여 라이선스 존재)

## Outcome
atomic: `contents.license_count` 차감 + `licenses` INSERT (status='active')

## Exception
- `E404-CONTENT`: 콘텐츠 미존재
- `E409-CONTENT`: 비공개/만료/archived 콘텐츠
- `E422-LICENSE-EXHAUSTED`: 라이선스 소진
