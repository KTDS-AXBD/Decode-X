# Runbook — AS-005 위성 일괄 퇴역 처리 batch

## 알림 조건
- inOrbit 위성 누적 (`payloads.status='inOrbit' AND deployed_at <= now - threshold`) 비정상 증가
- batch retire 후 통계 검증 — retiredCount=0 지속 시

## 진단
1. retire 기준 시각 (`now` 파라미터) 정확성 확인 (UTC vs local time drift)
2. 대상 위성 count 사전 측정 (배치 실행 전 cron pre-check)
3. batch run 후 retired_count 통계 비교 (전일 대비)

## 해소
- 즉시: 수동 batch 트리거 (cron miss 시) — `retireSatelliteBatch(now)` 단발 호출
- 영구: cron schedule 점검 + 모니터링 알림 추가 (inOrbit > threshold 시 alert)
