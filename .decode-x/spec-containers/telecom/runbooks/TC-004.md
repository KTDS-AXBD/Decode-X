# Runbook — TC-004: 가입 상태 전환

## 트리거
상태관리SYSTEM → `PATCH /api/v1/telecom/subscriptions/:id/status`

## 처리 흐름
1. 현재 상태 조회
2. 허용 전환 매트릭스 검증
3. `subscriptions.status` UPDATE + timestamp 갱신

## 허용 전환
- pending → active
- active → suspended
- suspended → active
- active → terminated
- suspended → terminated

## 모니터링
- E409-SUBSCRIPTION (비허용 전환 시도)
- 일시정지율 / 해지율
