# Runbook — MR-003: 통관 처리 atomic

## 트리거
세관직원/통관SYSTEM → `POST /api/v1/maritime/shipments/:id/customs`

## 처리 흐름
1. 선적 status='arrived' 확인
2. 관세 계산 (declaredValue × tariffRate)
3. atomic 트랜잭션:
   a. customs_declarations INSERT (status='cleared')
   b. customs_approvals INSERT
   c. shipments.customs_cleared=1 갱신

## 모니터링
- 통관 처리 시간 (declared_at → cleared_at)
- 관세 금액 분포
- E409-SHIPMENT 발생률 (비정상 상태 통관 시도)
