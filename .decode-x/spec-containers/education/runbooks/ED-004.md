# Runbook — ED-004: 수강 상태 전환

## 트리거
- 정상 수료 → completed
- 성적 입력 → graded
- 취소 → withdrawn
- 불합격 → failed

## 처리 흐름
1. 현재 status 조회
2. 허용 매트릭스 검증
3. `UPDATE enrollments SET status=?, completed_at/withdrawn_at`
4. `{previousStatus, newStatus}` 반환

## 알림
- graded 시 학생에게 성적 공지
- failed 시 학사 경고 트리거

## 모니터링
- 일별 상태 전환 카운트 per status
- 과목별 수료율 (completed / 총 enrolled)
