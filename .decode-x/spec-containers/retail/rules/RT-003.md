# RT-003 — 주문 체크아웃 Atomic (장바구니 + 재고 예약 + 결제 + 감사)

**BL ID**: RT-003
**Domain**: Retail (소매)
**Detector**: AtomicTransaction
**Function**: `processCheckout()`

## Condition
주문 체크아웃 처리 시

## Criteria
- `cartItems.length > 0` (장바구니 비어있지 않음)
- `totalAmount > 0` (금액 양수)

## Outcome
atomic: `orders` INSERT (status='placed') + `inventory_sync_log` INSERT

## Exception
- `E422-CART-EMPTY`: 장바구니 비어있음
- `E422-AMOUNT-MIN`: 결제 금액 0 이하
