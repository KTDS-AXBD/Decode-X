# CN-004 — 프로젝트 상태 전환

**BL ID**: CN-004
**Domain**: Construction (건설)
**Detector**: StatusTransition
**Function**: `transitionProjectStatus()`

## Condition
프로젝트 상태 전환 요청 시

## Criteria
허용 전환 매트릭스:
- bidding → awarded
- awarded → in_progress
- in_progress → completed
- completed → closed

## Outcome
`projects.status` UPDATE + 타임스탬프 기록 (awarded_at / started_at / completed_at / closed_at)

## Exception
- `E404-PROJECT`: 프로젝트 미존재
- `E409-PROJECT`: 허용되지 않는 상태 전환
