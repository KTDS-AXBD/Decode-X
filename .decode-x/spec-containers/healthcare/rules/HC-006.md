# HC-006: 예약 취소 + 환불 atomic

**Rule**: booked/checked_in 상태 검증 후 atomic하게 cancelled + slot 해제 (24시간 이전 시 100% 환불)

**구현 함수**: `cancelAppointmentWithRefund`
**Detector**: AtomicTransaction (`db.transaction(() => { ... }); tx();`)

## 트리거 조건
- **When**: 사용자 예약 취소 요청
- **If**: `appointment.status ∈ {booked, checked_in}`
- **Then**: atomic transaction
  1. `UPDATE appointments SET status='cancelled', cancelled_at=now`
  2. `UPDATE slots SET status='available'` (slot 해제)
- 환불 정책: 24시간 이전 → 100% (refundAmount = fee_krw), 미만 → 0원
- **Else**: `E404-AP` / `E409-AP`

## 무결성 보장
- 예약 취소 + slot 해제 atomic — slot 잠금 잔존 방지
- atomic rollback 시 두 변경 모두 무효화

## 환불 정책
- 24시간 이전 취소 → 100% 환불 (별도 결제 시스템 연동)
- 24시간 미만 → 0원 (취소 자체는 가능)

## 신뢰도
- Detector: 85% (AtomicTransaction)
