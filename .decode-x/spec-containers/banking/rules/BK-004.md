# BK-004 — 계좌 상태 전환

**BL ID**: BK-004
**Domain**: Banking (은행)
**Detector**: StatusTransition
**Function**: `transitionAccountStatus()`

## Condition
계좌 상태 전환 (pending_kyc → active → frozen → dormant → closed)

## Criteria
허용 전환 매트릭스:
- `pending_kyc → active` (KYC 완료)
- `active → frozen` (이상거래 동결)
- `active → dormant` (장기 미사용)
- `frozen → active` (동결 해제)
- `frozen → closed` (강제 해지)
- `dormant → active` (재활성화)
- `dormant → closed` (휴면 해지)

## Outcome
`accounts.status` UPDATE + 타임스탬프(`frozen_at`/`dormant_at`/`closed_at`) 기록

## Exception
- `E404-ACCOUNT`: 계좌 미존재
- `E409-ACCOUNT`: 허용 매트릭스 위반
