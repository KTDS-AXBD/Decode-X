# AV-001 — 승객 탑승 한도 검증

**함수**: `boardPassenger()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant)

## 조건
승객 탑승 요청 시 `passengerCount >= MAX_PASSENGER_CAPACITY` 검사.

## 기준
`MAX_PASSENGER_CAPACITY = 400` (항공기 최대 탑승 한도)

## 결과
- 한도 내: 탑승 허용, passenger 생성
- 한도 초과: `E422-CAPACITY-LIMIT` 예외

## 근거
항공 안전 운항 기준 — 정원 초과 탑승 금지.
