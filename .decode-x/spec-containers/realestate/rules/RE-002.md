# RE-002: 임차 한도 검증 (월세/소득 비율)

**Rule**: monthly_income × 0.4 ≥ monthlyRentKrw 충족 시 임차 가능

**구현 함수**: `checkRentAffordabilityLimit`
**Detector**: ThresholdCheck (F445 Path B — `rentLimit < monthlyRentKrw`, `limit` keyword)

## 트리거 조건
- **When**: 계약 체결 직전
- **If**: `tenant.status='verified'` AND `rentLimit ≥ monthlyRentKrw`
- **Then**: `{canLease: true, remainingRentLimit}` 반환
- **Else**: `E404-TN` / `E409-ST` / `E422-LIMIT`

## 신뢰도
- Detector: 70% (Path B — `limit` keyword)
