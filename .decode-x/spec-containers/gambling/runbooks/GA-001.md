# Runbook — GA-001 카지노 동시 active bet 한도

## 알림 조건
`E422-CASINO-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `casinos.active_bets` vs `total_capacity` 비교 조회
2. 특정 카지노/피크 시간대(주말 야간) 동시 베팅 집중 여부 확인
3. `MAX_CONCURRENT_ACTIVE_BETS_PER_CASINO` 기본값(200) 적절성 검토 (테이블/슬롯 머신 수 대비)

## 해소
- 임시: 카지노에 settled 후보 bet 정리 권유 또는 total_capacity 직접 조정
- 영구: 테이블/슬롯 증설 + waiting list 정책 명문화 + 책임도박 cool-down 정책
