# Runbook — RT-002: 프로모션 자격 최소 주문액 한도 비교

## 트리거
프로모션 적용 → `POST /api/v1/retail/promotions/:id/apply`

## 처리 흐름
1. promotionId로 프로모션 조회 + status 확인
2. minOrderLimit = max(promotion.min_order_amount, MIN_ORDER_AMOUNT)
3. cartTotal vs minOrderLimit 비교
4. 할인 금액 10% 계산

## 모니터링
- E422-PROMO-MIN 발생률 (최소 주문액 미달 비율)
- 프로모션별 전환율
