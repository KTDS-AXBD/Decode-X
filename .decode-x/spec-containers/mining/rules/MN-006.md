# MN-006 — 안전 사고 atomic

**함수**: `processSafetyIncident()`
**Detector**: AtomicTransaction

## 조건
안전 사고 신고 시 원자적 처리.

## 기준
사고 신고 필수 (incidentType + severity + description + correctiveAction)

## 결과
atomic: safety_incidents INSERT + incident_investigations INSERT + corrective_actions INSERT + safety_incidents.filed UPDATE

## 근거
산업안전보건법 — 중대재해 발생 시 사고 신고 + 조사 + 시정조치가 동시에 기록되어야 함.
