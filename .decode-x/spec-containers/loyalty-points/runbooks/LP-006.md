# Runbook — LP-006: 환불 회수

## 트리거
- 결제 환불 처리 시 trigger (적립 ledger ID 전달)

## 처리 흐름
1. EARN ledger 조회 → 미존재 시 E404-LEDGER
2. days_since_earn > 30 → E422-WINDOW throw
3. balance < earn_amount → E422-USED throw (이미 사용)
4. UPDATE balance/total_earned 차감 + INSERT CLAWBACK ledger

## 모니터링
