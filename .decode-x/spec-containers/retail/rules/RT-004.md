# RT-004 — 주문 상태 전환 (placed → confirmed → shipped → delivered → completed)

**BL ID**: RT-004
**Domain**: Retail (소매)
**Detector**: StatusTransition
**Function**: `transitionOrderStatus()`

## Condition
주문 상태 전환 처리 시

## Criteria
허용 전환 매트릭스:
- placed → confirmed
- confirmed → shipped
- shipped → delivered
- delivered → completed

## Outcome
`orders.status` UPDATE

## Exception
- `E404-ORD`: 주문 미존재
- `E409-ORD`: 허용되지 않는 상태 전환
