# TC-006 — 번호이동 atomic 트랜잭션

**BL ID**: TC-006
**Domain**: Telecom (통신)
**Detector**: AtomicTransaction
**Function**: `processPortOut()`

## Condition
번호이동 처리 요청 시

## Criteria
- `subscription.status != 'terminated'` (이미 해지된 회선 번호이동 불가)

## Outcome
atomic: `port_out_requests` INSERT (status='completed') + `subscriptions.status='terminated'` UPDATE + `terminated_at` 기록

## Exception
- `E404-SUBSCRIPTION`: 회선 미존재
- `E409-SUBSCRIPTION`: 이미 해지된 회선 번호이동 시도
