# Runbook — MD-002: 무료 시청 한도 비교

## 트리거
구독자/시청SYSTEM → `GET /api/v1/media/contents/:id/watch`

## 처리 흐름
1. 사용자 구독 tier 확인 (free/basic/standard/premium)
2. tier별 viewQuotaLimit 계산
3. `viewedCount > viewQuotaLimit` 차단
4. 시청 허용 + 카운트 증가

## 모니터링
- E422-VIEW-QUOTA 발생률 (한도 도달 free 구독자 비율)
- tier 업그레이드 전환율 (한도 도달 후 유료 전환)
