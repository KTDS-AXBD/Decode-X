# SP-002 — 시즌권 한도 비교

**함수**: `applySeasonTicketTier()`
**Detector**: ThresholdCheck (Path B, var-vs-var, `seasonTicketLimit` keyword)

## 조건
시즌권 신청 시 `requestedQuantity > seasonTicketLimit` 검사.

## 기준
`seasonTicketLimit = season_ticket_tiers.seasonTicketLimit` (단계별 시즌권 한도)

## 결과
- 한도 내: 시즌권 신청 승인, season_ticket_requests 생성
- 한도 초과: `E422-SEASON-TICKET-EXCEEDED` 예외

## 근거
공정 배분 원칙 — 특정 회원의 시즌권 독점 방지.
