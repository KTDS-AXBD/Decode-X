# CN-001 — 입찰 금액 한도 검증

**BL ID**: CN-001
**Domain**: Construction (건설)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `submitBid()`

## Condition
입찰 제출 시

## Criteria
- `bidAmount < MAX_BID_AMOUNT_LIMIT` (최대 입찰 한도 1,000,000,000 KRW)
- 프로젝트 status='bidding' 확인

## Outcome
입찰 허용 + bids INSERT

## Exception
- `E422-BID-LIMIT`: 입찰 금액 한도 초과 (1,000,000,000 KRW)
- `E409-PROJECT`: 비정상 프로젝트 상태
- `E404-PROJECT`: 프로젝트 미존재
