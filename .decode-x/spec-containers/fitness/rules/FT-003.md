# FT-003 — 퍼스널 트레이닝 예약 Atomic

**함수**: `bookPersonalTraining()`
**Detector**: AtomicTransaction

## 조건
PT 예약 요청 시 트레이너 슬롯 available 상태 검사.

## 기준
`trainer_slots.status = 'available'`

## 결과
- 슬롯 가용: atomic 트랜잭션 — pt_bookings INSERT + trainer_slots UPDATE + pt_payments INSERT
- 슬롯 불가: `E404-SLOT` 예외

## 근거
PT 예약, 트레이너 슬롯 점유, 결제 처리를 단일 트랜잭션으로 보장.
