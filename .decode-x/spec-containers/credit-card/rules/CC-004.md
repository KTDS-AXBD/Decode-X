# CC-004: 카드 상태 전환 (active → blocked / expired / cancelled)

**Rule**: active 상태 카드만 다른 상태(blocked/expired/cancelled)로 전환 가능

**구현 함수**: `transitionCardStatus`
**Detector**: StatusTransition (`status` 필드 read-then-write 패턴)

## 트리거 조건
- **When**: 카드 상태 전환 요청 (관리자 차단 / 만료 batch / 사용자 해지)
- **If**: `previousStatus = 'active'` AND `newStatus ∈ {blocked, expired, cancelled}`
- **Then**: `UPDATE credit_cards SET status = ?` + `{previousStatus, newStatus}` 반환
- **Else**: `E409-TR` (active 아닌 상태에서 전환 불가)

## 상태 머신
```
active → blocked  (관리자 차단)
active → expired  (expires_at 도달)
active → cancelled (사용자 해지)
```

## 신뢰도
- Detector: 75% (StatusTransition — SELECT status + UPDATE status 동시)
