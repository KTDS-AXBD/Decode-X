# BK-003 — 계좌 이체 atomic

**BL ID**: BK-003
**Domain**: Banking (은행)
**Detector**: AtomicTransaction
**Function**: `processAccountTransfer()`

## Condition
계좌 이체 요청 시

## Criteria
- `fromAccount.status = 'active'`
- `toAccount.status = 'active'`
- `fromAccount.balance >= amount + fee`

## Outcome
atomic: `accounts.balance` 출금 계좌 차감 + 입금 계좌 증가 + `transactions` INSERT

## Exception
- `E404-ACCOUNT`: 계좌 미존재
- `E409-ACCOUNT`: 비활성 계좌
- `E422-INSUFFICIENT-FUNDS`: 잔액 부족 (수수료 포함)
