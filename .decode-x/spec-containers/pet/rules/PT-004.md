# PT-004 — 케어 상태 전환

**함수**: `transitionCareStatus()`
**Detector**: StatusTransition

## 조건
케어 상태 전환 요청 시 허용 매트릭스 확인.

## 허용 전환
```
booked → checked_in
checked_in → in_care
in_care → checked_out
checked_out → reviewed
```

## 결과
허용 시: `care_records.status` UPDATE

## 예외
- `E404-CARE-RECORD`: 기록 미존재
- `E409-CARE-RECORD`: 허용되지 않는 전환

## 근거
반려동물 케어 서비스 lifecycle 상태 일관성 보장 — 순서 외 전환 차단.
