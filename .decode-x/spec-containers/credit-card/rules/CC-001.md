# CC-001: 카드 발급 신용평가

**Rule**: 신용점수 ≥ 600 AND 연소득 ≥ 24,000,000 KRW 충족 시 카드 발급, 미충족 시 거절

**구현 함수**: `issueCard`
**Detector**: ThresholdCheck (`creditScore < MIN_CREDIT_SCORE` || `annualIncome < MIN_INCOME_KRW`)

## 트리거 조건
- **When**: 카드 발급 신청 접수 시
- **If**: `creditScore ≥ 600` AND `annualIncome ≥ 24,000,000`
- **Then**: `credit_cards` INSERT (status='active', credit_limit = min(requested, baseLimit + scoreBonus))
- **Else**: `E422-CS` (신용점수 미달) 또는 `E422-IN` (연소득 미달)

## 한도 산정
- `baseLimit = floor(annualIncome × 0.3)`
- `scoreBonus`: 750↑ +5M / 700↑ +2M / else 0
- `approvedLimit = min(requestedLimit, baseLimit + scoreBonus)`

## 신뢰도
- Detector: 70% (ThresholdCheck — `<` 비교 + 이중 검증)
