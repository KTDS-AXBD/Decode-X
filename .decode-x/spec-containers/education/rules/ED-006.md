# ED-006: 수강 취소 + 환불 atomic

**Rule**: enrolled 검증 후 atomic하게 withdrawn + course enrolled_count-- (학기 시작 7일 이전 시 100% 환불)

**구현 함수**: `withdrawEnrollmentWithRefund`
**Detector**: AtomicTransaction (`db.transaction(() => { ... }); tx();`)

## 트리거 조건
- **When**: 학생 수강 취소 요청
- **If**: `enrollment.status='enrolled'`
- **Then**: atomic transaction
  1. `UPDATE enrollments SET status='withdrawn', withdrawn_at=now`
  2. `UPDATE courses SET enrolled_count = enrolled_count - 1`
- 환불 정책: 학기 시작 7일 이전 → 100% (refundAmount = fee_krw), 미만 → 0원
- **Else**: `E404-EN` / `E409-EN`

## 무결성 보장
- 취소 + course 자리 해제 atomic — capacity 누적 정합 보장

## 신뢰도
- Detector: 85% (AtomicTransaction)
