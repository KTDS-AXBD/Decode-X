# HC-002: 처방 일일 dosage 한도 검증

**Rule**: `MAX_DAILY_DOSAGE_MG - activeDosage ≥ requestedDosageMg` 충족 시 처방 가능

**구현 함수**: `checkDosageLimit`
**Detector**: ThresholdCheck (F445 Path B — `dosageLimit < requestedDosageMg`, `limit` keyword 매칭)

## 트리거 조건
- **When**: 의사 처방 직전
- **If**: `patient.status='active'` AND `dosageLimit ≥ requestedDosageMg`
- **Then**: `{canPrescribe: true, remainingDosageHeadroom}` 반환
- **Else**: `E404-PT` / `E409-ST` / `E422-LIMIT`

## 신뢰도
- Detector: 70% (Path B — `limit` keyword)
