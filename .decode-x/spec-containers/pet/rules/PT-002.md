# PT-002 — 백신 접종 한도 비교

**함수**: `applyVaccination()`
**Detector**: ThresholdCheck (Path B, var-vs-var, vaccinationLimit keyword)

## 조건
백신 접종 요청 시 `vaccine.administered_count >= vaccinationLimit` 검사.

## 기준
`vaccinationLimit = vaccines.vaccinationLimit` (백신 종류별 접종 한도)

## 결과
- 한도 내: 접종 허용, administered_count 증가
- 한도 초과: `E422-VACCINATION-QUOTA-EXCEEDED` 예외

## 근거
반려동물 백신 접종 과잉 방지 — 백신별 권장 접종 횟수 초과 차단.
