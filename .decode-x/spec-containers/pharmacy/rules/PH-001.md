# PH-001 — 일일 최대 용량 한도 검증

**BL ID**: PH-001
**Domain**: Pharmacy (제약/약국)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `validateDosage()`

## Condition
처방 용량 검증 시

## Criteria
- `dosageAmount < MAX_DAILY_DOSE` (일일 최대 4000mg)
- 처방 status='issued' or 'pending' 확인

## Outcome
용량 허용 + status='pending' 전환

## Exception
- `E422-DOSAGE-EXCEEDED`: 일일 최대 용량 초과 (4000mg)
- `E409-PRESCRIPTION`: 비정상 처방 상태
- `E404-PRESCRIPTION`: 처방 미존재
