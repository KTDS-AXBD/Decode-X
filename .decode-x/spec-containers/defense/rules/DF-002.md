# DF-002 — 보안 레벨 한도 비교

**함수**: `checkClearanceLevel()`
**Detector**: ThresholdCheck (Path B, var-vs-var)

## 조건
보안 레벨 검증 시 `clearanceLevel > clearanceLevelLimit` 검사.

## 기준
`clearanceLevelLimit = clearance_tiers.clearance_level_limit` (단계별 최대 보안 레벨)

## 결과
- 한도 내: 보안 레벨 승인
- 한도 초과: `E422-CLEARANCE-EXCEEDED` 예외
- 필요 레벨 미달: `E403-CLEARANCE-INSUFFICIENT` 예외

## 근거
국방 보안 규정 — 단계별 보안 레벨 상한 준수 의무.
