# CH-006 — 세금증명 발급 atomic

**함수**: `issueTaxCertificate()`
**Detector**: AtomicTransaction

## 조건
세금증명 발급 요청 시 기부 내역 존재 필수.

## 기준
`donations` 레코드 존재 검증

## 결과
- 정상: `tax_certificates` INSERT + `donations` UPDATE + `tax_reports` INSERT + `tax_certificates.reported_at` UPDATE (atomic)
- 실패 시: CharityError

## 근거
세금증명 발급은 발급→등록→신고가 동시에 이루어져야 함 — 중간 실패 시 국세청 신고 오류 방지.
