# MF-001 — BOM 폭발 구성품 수 한도 검증

**BL ID**: MF-001
**Domain**: Manufacturing (제조)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `explodeBom()`

## Condition
BOM 폭발 요청 시

## Criteria
- `components.length ≤ BOM_MAX_COMPONENTS` (최대 500개 구성품)
- `components.length > 0` (비어있지 않음)

## Outcome
`boms` 테이블 INSERT (status='active')

## Exception
- `E422-BOM-MAX`: 최대 구성품 수 초과
- `E422-BOM-EMPTY`: 구성품 목록 비어있음
