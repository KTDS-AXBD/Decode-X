# WL-002 — 멤버십 패키지 사용 한도 비교

**함수**: `usePackageSession()`
**Detector**: ThresholdCheck (Path B, var-vs-var, packageUsageLimit keyword)

## 조건
패키지 세션 사용 요청 시 `pkg.used_count >= packageUsageLimit` 검사.

## 기준
`packageUsageLimit = session_packages.packageUsageLimit` (패키지 티어별 사용 가능 횟수)

## 결과
- 한도 내: 사용 허용, session_packages.used_count 증가
- 한도 초과: `E422-PACKAGE-USAGE-EXCEEDED` 예외

## 근거
멤버십 패키지 계약 상 포함 세션 횟수 초과 방지. 티어(basic/premium/elite/vip)별 한도 차등 적용.
