# MR-003 — 통관 처리 atomic

**BL ID**: MR-003
**Domain**: Maritime (해운)
**Detector**: AtomicTransaction
**Function**: `processCustoms()`

## Condition
통관 처리 시

## Criteria
- `shipment.status = 'arrived'`
- 관세 계산 = declaredValue × tariffRate

## Outcome
atomic: customs_declarations INSERT + customs_approvals INSERT + shipments.customs_cleared 갱신

## Exception
- `E404-SHIPMENT`: 선적 미존재
- `E409-SHIPMENT`: 비정상 선적 상태 (arrived가 아닌 경우)
