# FT-006 — 기구 예약 Atomic

**함수**: `reserveEquipment()`
**Detector**: AtomicTransaction

## 조건
기구 예약 요청 시 available 상태 검사.

## 기준
`equipment.status = 'available'`

## 결과
- 기구 가용: atomic 트랜잭션 — equipment_reservations INSERT + equipment_holds INSERT + equipment UPDATE
- 기구 불가: `E404-EQUIPMENT` 예외

## 근거
피트니스 기구 예약, hold 기록, 일일 사용 통계를 단일 트랜잭션으로 보장.
