# CH-004 — 캠페인 상태 전환

**함수**: `transitionCampaignStatus()`
**Detector**: StatusTransition

## 조건
캠페인 상태 전환 요청 시 허용 매트릭스 검사.

## 상태 머신
`draft → active → closed → reported → audited`

## 결과
- 허용 전환: `campaigns.status` UPDATE + 타임스탬프 기록
- 불허 전환: `E409-CAMPAIGN` 예외
- 캠페인 없음: `E404-CAMPAIGN` 예외

## 근거
비영리단체 캠페인 라이프사이클 규정 — 준비→모집→마감→보고→감사 순서 준수 의무.
