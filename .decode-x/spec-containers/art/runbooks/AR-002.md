# Runbook — AR-002 컬렉터 일일 acquisition 한도

## 알림 조건
`E422-DAILY-ACQUISITION-LIMIT-EXCEEDED` 빈도 급증 시.

## 진단
1. `collector_contracts.acquisition_used` vs `acquisition_limit` 비교
2. 등급(free/bronze/silver/gold/platinum)별 한도 분포 점검
3. acquisition 카운트 측정 정확성 확인 (1점 vs N점 묶음 거래) — acquisition 인플레이션 의심 시 audit

## 해소
- 임시: 등급 업그레이드 또는 promo 제공 (보너스 acquisition)
- 영구: `acquisition_limit` 등급 재조정 또는 daily reset cron 정상 동작 + 묶음 거래 정책 명문화
