# Runbook — EN-001: 계량기 검침 사용량 한도 검증

## 트리거
검침원 → `POST /api/v1/energy/meters/{id}/readings`

## 처리 흐름
1. usageKwh 유효성 검증 (음수 / MAX_METER_USAGE_KWH 초과 차단)
2. 계량기 상태 확인 (suspended/locked 차단)
3. `meter_readings` INSERT
4. `meters.last_reading_kwh` UPDATE

## 모니터링
- E422-USAGE-MAX 발생률 (피크 사용량 이상 신호)
- E409-METER 발생률 (정지 계량기 비율)
