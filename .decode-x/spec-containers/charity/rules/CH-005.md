# CH-005 — 자원봉사 일정 일괄 갱신

**함수**: `markVolunteerSchedule()`
**Detector**: StatusTransition (batch 패턴, CC-005 27번째 재사용)

## 조건
일정 기준일 이전 pending 자원봉사 일정 일괄 처리.

## 기준
`volunteer_schedules.scheduled_date <= scheduledBefore AND status = 'pending'`

## 결과
- 대상 있음: `status='assigned', synced_at=NOW()` 일괄 UPDATE
- 대상 없음: syncedCount=0

## 근거
자원봉사 배치 자동화 — 예정일 도래 시 일괄 배정으로 관리 효율화.
