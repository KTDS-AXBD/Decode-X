# HO-001 — 객실 예약 한도 검증

**BL ID**: HO-001
**Domain**: Hospitality (숙박)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `bookRoom()`

## Condition
객실 예약 요청 시

## Criteria
- `requestedRooms ≤ availableRooms` (가용 객실 이내)
- `requestedRooms ≤ MAX_ROOMS_PER_BOOKING` (1회 최대 10실)
- `requestedRooms > 0` (양수 조건)

## Outcome
`bookings` 테이블 INSERT (status='pending')

## Exception
- `E422-RM-MAX`: 최대 예약 객실 초과
- `E422-RM-AVAIL`: 가용 객실 부족
- `E422-RM-MIN`: 0 이하 객실 수
