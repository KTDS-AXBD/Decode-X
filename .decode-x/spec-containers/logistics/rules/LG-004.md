# LG-004 — 배송 상태 전환

**BL ID**: LG-004
**Domain**: Logistics (물류)
**Detector**: StatusTransition
**Function**: `transitionDeliveryStatus()`

## Condition
배송 상태 변경 요청 시

## Criteria (허용 매트릭스)
- pending → in_transit
- in_transit → delivered
- in_transit → returned

## Outcome
`shipments.status` UPDATE + delivered_at 조건부 기록

## Exception
- `E404-SH`: 화물 미존재
- `E409-TR`: 허용되지 않은 상태 전환
