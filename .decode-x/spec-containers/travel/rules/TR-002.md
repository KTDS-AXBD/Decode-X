# TR-002 — 운임 등급 업그레이드 마일리지 한도 비교

**BL ID**: TR-002
**Domain**: Travel (여행)
**Detector**: ThresholdCheck (Path B — var-vs-var, `requiredMilesLimit` 'limit' keyword 매칭)
**Function**: `upgradeFareClass()`

## Condition
운임 등급 업그레이드 요청 시 (itinerary.status='pending' or 'confirmed')

## Criteria
- `availableMiles ≥ requiredMilesLimit` (보유 마일리지 충분)

## Outcome
`fare_class` UPDATE + `miles_used` 누적

## Exception
- `E404-IT`: 여정 미존재
- `E409-IT`: 업그레이드 불가 상태
- `E422-MILES`: 마일리지 부족
