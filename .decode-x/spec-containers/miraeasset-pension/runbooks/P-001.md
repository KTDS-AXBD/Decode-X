# Runbook — P-001: 가입 자격 검증

**Rule**: 근속기간 ≥ 1년 AND 연령 ≥ 18세 AND 고용 상태 = 재직 중이면 가입 승인

## 트리거
- 가입자가 `POST /api/v1/pension/enroll` 호출

## 전제조건
- `yearsOfService`: 근속연수 (회사 HR 시스템 조회)
- `age`: 신청자 연령 (주민등록번호 기반)
- `employmentStatus`: 고용 상태 (`ACTIVE` | `RESIGNED`)

## 처리 흐름
1. `yearsOfService < 1` → `E422-INELIGIBLE` 반환
2. `age < 18` → `E422-INELIGIBLE` 반환
3. `employmentStatus !== 'ACTIVE'` → `E422-INELIGIBLE` 반환
4. 모든 조건 통과 시 → `pension_accounts` INSERT + `PensionEnrolled` 이벤트 발행
