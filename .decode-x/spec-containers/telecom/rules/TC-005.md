# TC-005 — 청구 주기 일괄 처리 (배치)

**BL ID**: TC-005
**Domain**: Telecom (통신)
**Detector**: StatusTransition (batch 패턴 — CC-005 batch 15번째 재사용)
**Function**: `runBillingCycle()`

## Condition
청구 주기 일괄 처리 (배치) 실행 시

## Criteria
- `billing_cycles.status = 'pending'` AND 해당 청구 월(`cycle_month`) 대상 확인

## Outcome
`billing_cycles.status='billed'` 일괄 UPDATE + `billed_at` 기록

## Exception
- 대상 없으면 billedCount=0 (정상 종료)
