# Runbook — BR-001 방송국 동시 active broadcast 한도

## 알림 조건
`E422-STATION-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `stations.active_broadcasts` vs `total_capacity` 비교 조회
2. 특정 방송국/채널/시즌(올림픽, 선거, 긴급 보도) 동시 편성 집중 여부 확인
3. `MAX_CONCURRENT_ACTIVE_BROADCASTS_PER_STATION` 기본값(24) 적절성 검토 (단일 채널 vs 다채널 종편)

## 해소
- 임시: 방송국에 archived 후보 broadcast 정리 권유 또는 total_capacity 직접 조정 (종편 상향)
- 영구: 채널 분할 정책 명문화 + 편성 인프라 증설 (24h+ 가능한 멀티 채널)
