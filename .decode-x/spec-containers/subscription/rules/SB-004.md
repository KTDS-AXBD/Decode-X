# SB-004: 구독 상태 전환 (active ↔ paused, → cancelled)

**Rule**: 허용된 상태 전환만 인정. 종료 상태(cancelled/expired)에서는 전환 불가

**구현 함수**: `transitionSubscriptionStatus`
**Detector**: StatusTransition (`status` SELECT-then-UPDATE 패턴)

## 트리거 조건
- **When**: 사용자 일시 중지/재개/취소 요청
- **If**: 허용 매트릭스 충족
  - `active → paused` (사용자 일시 중지)
  - `active → cancelled` (사용자 취소, 환불 없이 SB-004로만)
  - `paused → active` (재개)
  - `paused → cancelled` (일시 중지 중 취소)
- **Then**: `UPDATE subscriptions SET status=?, cancelled_at=?` (cancelled 시점만 cancelled_at 설정)
- **Else**: `E409-TR` (불허 전환)

## 상태 머신
```
active ↔ paused
active → cancelled
paused → cancelled
```

## 신뢰도
- Detector: 75% (StatusTransition — SELECT status + UPDATE status)
