# Runbook — AG-002: 살충제 살포 한도 비교

## 트리거
농부/농약관리SYSTEM → `POST /api/v1/agriculture/fields/:id/pesticide`

## 처리 흐름
1. field.status != 'restricted' 확인 (제한 구역 차단)
2. 면적 카테고리 결정 (small/medium/large)
3. `pesticideApplied > pesticideQuotaLimit` 초과 차단
4. 허용 시 pesticide_logs INSERT

## 모니터링
- E422-PESTICIDE-QUOTA 발생률 (허용치 초과 살포 시도)
- 제한 구역 살포 시도 횟수 (E409-FIELD)
