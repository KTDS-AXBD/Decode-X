# BT-002 — 로열티 멤버십 할인 한도 비교

**함수**: `applyLoyaltyDiscount()`
**Detector**: ThresholdCheck (Path B, var-vs-var, loyaltyTierLimit keyword 매칭)

## 조건
로열티 할인 적용 요청 시 `membership.loyalty_usage >= loyaltyTierLimit` 검사.

## 기준
`loyaltyTierLimit = loyalty_memberships.loyalty_tier_limit` (멤버십 등급별 할인 한도)

## 결과
- 한도 내: 할인 허용, loyalty_usage 증가
- 한도 초과: `E422-LOYALTY-TIER-LIMIT-EXCEEDED` 예외

## 근거
미용실 멤버십 등급(bronze/silver/gold/platinum)별 월별 할인 사용 한도 관리.
