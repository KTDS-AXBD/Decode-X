# MD-001 — 구독 tier 동시 시청 한도 검증

**BL ID**: MD-001
**Domain**: Media (미디어)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `activateMediaSubscription()`

## Condition
스트리밍 요청 시

## Criteria
- `concurrentStreamCount < MAX_CONCURRENT_STREAMS` (최대 4 동시 스트림)
- 구독 status='active' 확인

## Outcome
스트리밍 허용 + 세션 활성화

## Exception
- `E422-STREAM-LIMIT`: 동시 시청 한도 초과 (4 스트림)
- `E409-SUBSCRIPTION`: 비활성 구독
- `E404-SUBSCRIPTION`: 구독 미존재
