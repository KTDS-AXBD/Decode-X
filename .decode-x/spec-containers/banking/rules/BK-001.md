# BK-001 — 일일 출금 한도 검증

**BL ID**: BK-001
**Domain**: Banking (은행)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `processWithdrawal()`

## Condition
출금 요청 시

## Criteria
- `amount < MAX_WITHDRAWAL_AMOUNT` (최대 1천만원/건)
- 계좌 status='active' 확인
- 잔액(`accounts.balance`) 충분 확인

## Outcome
`transactions` INSERT + `accounts.balance` 차감

## Exception
- `E422-WITHDRAWAL-LIMIT`: 1회 출금 한도 초과 (1천만원)
- `E422-INSUFFICIENT-FUNDS`: 잔액 부족
- `E409-ACCOUNT`: 비활성 계좌
