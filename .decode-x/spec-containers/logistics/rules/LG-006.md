# LG-006 — 반품 RMA + 재고 복구 Atomic

**BL ID**: LG-006
**Domain**: Logistics (물류)
**Detector**: AtomicTransaction
**Function**: `processReturnRma()`

## Condition
반품 RMA 요청 시 (shipment.status='delivered' or 'in_transit')

## Criteria
- 발송일 기준 30일 이내

## Outcome
Atomic: `rma_records` INSERT (status='initiated') + `shipments.status='returned'`

## Exception
- `E404-SH`: 화물 미존재
- `E409-SH`: 상태 불일치 (반품 불가 상태)
- `E422-RMA-EXP`: 반품 가능 기간 초과 (30일 경과)
