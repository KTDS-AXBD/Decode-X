# SP-005 — 상품 일괄 동기화 처리

**함수**: `markMerchandiseSync()`
**Detector**: StatusTransition (batch 패턴, CC-005 26번째 재사용)

## 조건
배치 동기화 시 `created_at <= syncedBefore AND status = 'pending'` 해당 항목 일괄 처리.

## 기준
처리 대상: `merchandise_batches.status = 'pending'` + `created_at <= syncedBefore`.

## 결과
- 대상 있음: `status='synced', synced_at=NOW()` 일괄 UPDATE
- 대상 없음: syncedCount=0 반환

## 근거
상품 재고 동기화 — 이벤트 당일 판매 상품 재고 상태 일괄 갱신.
