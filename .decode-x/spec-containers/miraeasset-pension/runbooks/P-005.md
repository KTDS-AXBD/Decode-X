# Runbook — P-005: 세액공제 적용

**Rule**: 납입액 ≤ 900만원 한도 이내분에 세액공제 처리

## 트리거
- `POST /api/v1/pension/{accountId}/tax-benefit`

## 전제조건
- `annualContribution`: 해당 역년 실제 납입 금액
- `TAX_BENEFIT_LIMIT_KRW = 9_000_000`

## 처리 흐름
1. `annualContribution <= 0` → `E422-INVALID_AMOUNT` 반환
2. `eligibleAmount = Math.min(annualContribution, TAX_BENEFIT_LIMIT_KRW)`
3. 한도 초과분(`annualContribution - eligibleAmount`) → 세액공제 제외 기록
4. `TaxBenefitApplied` 이벤트 발행 (eligible_amount 포함)
