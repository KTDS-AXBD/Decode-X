# Runbook — GV-002: 누진 수수료 구간 계산

## 트리거
수수료 SYSTEM → `POST /api/v1/government/permits/{id}/fee`

## 처리 흐름
1. `fee_tiers` 구간 조회 (tier_level ASC)
2. `feeAmount <= feeTierLimit` 매칭 구간 선택
3. `computedFee = feeAmount × rate_per_unit` 산출
4. `permit_applications.status='reviewing'` 갱신

## 모니터링
- E404-FEE-TIER 발생 시 fee_tiers 설정 점검
