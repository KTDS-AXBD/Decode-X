# MF-004 — 생산 상태 전환

**BL ID**: MF-004
**Domain**: Manufacturing (제조)
**Detector**: StatusTransition
**Function**: `transitionProductionStatus()`

## Condition
생산 lot 상태 전환 요청 시

## Criteria
허용 전환 매트릭스:
- `planned → in_progress` (생산 시작)
- `in_progress → qc` (품질 검사 진입)
- `qc → released` (출하 승인)

## Outcome
`production_lots.status` UPDATE

## Exception
- `E404-LOT`: 생산 lot 미존재
- `E409-LOT`: 허용되지 않는 상태 전환
