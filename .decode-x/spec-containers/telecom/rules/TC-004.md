# TC-004 — 가입 상태 전환 (pending → active → suspended → terminated)

**BL ID**: TC-004
**Domain**: Telecom (통신)
**Detector**: StatusTransition
**Function**: `transitionSubscriptionStatus()`

## Condition
가입 상태 전환 요청 시

## Criteria
허용 전환 매트릭스:
- `pending → active` (가입 활성화)
- `active → suspended` (일시 정지)
- `suspended → active` (정지 해제)
- `active → terminated` (해지)
- `suspended → terminated` (정지 중 해지)

## Outcome
`subscriptions.status` UPDATE + timestamp 필드 갱신 (activated_at / suspended_at / terminated_at)

## Exception
- `E404-SUBSCRIPTION`: 회선 미존재
- `E409-SUBSCRIPTION`: 허용되지 않은 상태 전환
