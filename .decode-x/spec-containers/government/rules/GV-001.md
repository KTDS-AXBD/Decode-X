# GV-001 — 허가 신청 연간 한도 검증

**BL ID**: GV-001
**Domain**: Government (공공/정부)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `submitPermitApplication()`

## Condition
허가 신청 요청 시

## Criteria
- `currentCount < MAX_ANNUAL_PERMIT_COUNT` (최대 10건/회계연도)
- 회계연도(`fiscal_year`) 기준 기존 신청 건수 조회

## Outcome
`permit_applications` INSERT + 신청 건수 증가

## Exception
- `E422-PERMIT-LIMIT`: 연간 허가 신청 한도 초과 (10건)
