# FT-005 — 노쇼 일괄 처리

**함수**: `markNoShowBatch()`
**Detector**: StatusTransition (batch 패턴)

## 조건
클래스 스케줄 기준일 이전 미출석 예약 일괄 처리.

## 기준
`fitness_classes.scheduled_at <= scheduledBefore` AND `class_bookings.status = 'booked'`

## 결과
- 해당 예약: `status = 'no_show'` 일괄 UPDATE
- 해당 없음: markedCount = 0

## 근거
피트니스 클래스 노쇼 관리 — 배치 처리로 패널티 부과 및 통계 집계.
