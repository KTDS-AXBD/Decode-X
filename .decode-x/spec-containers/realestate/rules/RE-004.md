# RE-004: 계약 상태 전환 (active → expiring/terminated/renewed)

**Rule**: 허용된 상태 전환만 인정

**구현 함수**: `transitionLeaseStatus`
**Detector**: StatusTransition

## 트리거 조건
- **When**: 만료 임박/해지/갱신 사유
- **If**: 허용 매트릭스
  - `active → expiring` (RE-005 batch 또는 manual)
  - `active → terminated` (RE-004 또는 RE-006)
  - `active → renewed` (RE-004 갱신)
  - `expiring → terminated/renewed` (RE-004)
- **Then**: `UPDATE leases SET status=?, terminated_at=?` (terminated 시점만)
- **Else**: `E409-TR`

## 상태 머신
```
active → expiring → terminated / renewed
active → terminated / renewed
```

## 신뢰도
- Detector: 75%
