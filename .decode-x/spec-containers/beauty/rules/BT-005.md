# BT-005 — 재고 재입고 일괄 처리

**함수**: `markInventoryRestockBatch()`
**Detector**: StatusTransition (batch 패턴, CC-005 32번째 재사용)

## 조건
재고 재입고 일괄 처리 시 depleted 항목 중 restocked_at 기준일 이전 대상 선별.

## 기준
`inventory_items.status = 'depleted'` AND `restocked_at <= restockedBefore`

## 결과
- 대상: status='restocked' 일괄 UPDATE
- 없음: markedCount=0 반환

## 근거
미용 재료(염색약, 케어제품 등) 일괄 재입고 처리 — 배치 방식으로 재고 상태 갱신.
