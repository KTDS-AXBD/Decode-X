# SP-004 — 이벤트 상태 전환

**함수**: `transitionEventStatus()`
**Detector**: StatusTransition

## 조건
이벤트 상태 전환 요청 시 허용 매트릭스 검사.

## 기준
허용 전환: `scheduled→ticketing`, `ticketing→live`, `live→completed`, `completed→archived`.

## 결과
- 허용: events.status UPDATE + 타임스탬프 기록
- 불허: `E409-EVENT` 예외

## 근거
이벤트 라이프사이클 관리 — 비순차 전환으로 인한 운영 오류 방지.
