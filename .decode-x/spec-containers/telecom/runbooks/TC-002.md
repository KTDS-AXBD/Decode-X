# Runbook — TC-002: 데이터 사용량 한도 비교

## 트리거
사용량SYSTEM → `POST /api/v1/telecom/subscriptions/:id/usage`

## 처리 흐름
1. 회선(`subscription_id`) 기준 요금제 `data_quota_limit` 조회
2. 누적 `usageBytes`와 비교
3. `usageBytes > dataQuotaLimit` 차단 (E429-DATA-QUOTA)
4. `data_usages` INSERT + 누적 기록

## 모니터링
- E429-DATA-QUOTA 발생률 (데이터 소진 고객 비율)
