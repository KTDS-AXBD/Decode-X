# Runbook — GV-006: 문서 검증 Atomic

## 트리거
문서 SYSTEM → `POST /api/v1/government/documents/{id}/validate`

## 처리 흐름
1. `document.status = 'pending'` 확인
2. atomic transaction:
   - `documents.status='validated'` + `validated_at` SET
   - `documents.status='certified'` + `certified_at` SET
   - `documents.status='issued'` + `issued_at` SET

## 모니터링
- E409-DOCUMENT 발생률 (중복 검증 시도)
