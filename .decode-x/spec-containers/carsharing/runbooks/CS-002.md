# Runbook — CS-002 회원 거리 사용 한도

## 알림 조건
`E422-DISTANCE-LIMIT-EXCEEDED` 빈도 급증 시.

## 진단
1. `member_passes.distance_used` vs `distance_limit` 비교 조회
2. 특정 tier_code 회원 비중 확인 (basic이 가장 자주 차단되는지)
3. 만료 임박 회원 잔여 거리 분포 조회

## 해소
- 임시: 회원 일대일 협의로 한도 추가 또는 상위 tier 업그레이드 권유
- 영구: tier별 distance_limit 정책 재설계 (basic 200km / standard 500km / premium 무제한 등)
