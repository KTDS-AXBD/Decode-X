# TS-001 — 노선 정원 한도 검증

**함수**: `checkRouteCapacity()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant)

## 조건
승객 탑승 요청 시 `passengerCount >= MAX_ROUTE_CAPACITY` 검사.

## 기준
`MAX_ROUTE_CAPACITY = 1,200` (노선 최대 정원)

## 결과
- 정원 내: 탑승 허용, trip 생성
- 정원 초과: `E422-CAPACITY-LIMIT` 예외

## 근거
대중교통 안전 운행 기준 — 정원 초과 승차 금지.
