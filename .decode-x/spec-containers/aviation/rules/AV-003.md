# AV-003 — 비행 운항 dispatch atomic

**함수**: `dispatchFlight()`
**Detector**: AtomicTransaction

## 조건
비행 운항 dispatch 시 스케줄+승무원+연료+이륙 허가 동시 처리.

## 기준
`flight.status = 'boarding'`

## 결과
atomic 트랜잭션:
1. `dispatch_records` INSERT (dispatch 이력)
2. `flight_clearances` INSERT (이륙 허가)
3. `flights.status = 'departed'` UPDATE
4. `crew_schedules.status = 'on_duty'` UPDATE

## 근거
비행 운항 dispatch는 원자적 완결 — 부분 실행 금지.
