# TS-003 — 환승 처리 atomic

**함수**: `processTransfer()`
**Detector**: AtomicTransaction

## 조건
`in_transit` 상태 트립에서 환승 요청.

## 기준
환승 등록 + 잔액 차감 + 통합권 발급 3단계 원자적 처리.

## 결과
단일 트랜잭션: transfers INSERT + integrated_passes INSERT + trips UPDATE(transferred) + 신규 trips INSERT

## 예외
`E409-TRIP` — in_transit 상태가 아닌 경우
