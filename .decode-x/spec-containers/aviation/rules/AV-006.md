# AV-006 — 수하물 클레임 atomic

**함수**: `processBaggageClaim()`
**Detector**: AtomicTransaction

## 조건
수하물 클레임 처리 시 매칭+손상검증+보상 동시 처리.

## 기준
`baggage.damage_status != 'intact'`

## 결과
atomic 트랜잭션:
1. `baggage_claims` INSERT (클레임 등록)
2. `damage_assessments` INSERT (손상 평가)
3. `compensation_records` INSERT (보상 기록)
4. `baggage_claims.baggage_claim_filed = 1` UPDATE

## 근거
수하물 클레임 처리는 원자적 완결 — 보상 미지급 상태 방지.
