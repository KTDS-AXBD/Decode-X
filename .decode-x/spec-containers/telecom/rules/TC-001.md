# TC-001 — 가입 회선 활성화 동시 한도 검증

**BL ID**: TC-001
**Domain**: Telecom (통신)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `activateSubscription()`

## Condition
가입 회선 활성화 요청 시

## Criteria
- `activeLines < MAX_ACTIVE_LINES` (최대 5회선/고객)
- 고객(`customer_id`) 기준 활성(status='active') 회선 건수 조회

## Outcome
`subscriptions` INSERT + status='active' 활성화 처리

## Exception
- `E422-LINE-LIMIT`: 동시 가입 회선 한도 초과 (5회선)
