# HC-005: 약제 만료 자동 처리

**Rule**: active 처방 + expires_at 도달 시 status='expired' 자동 마킹

**구현 함수**: `markExpiredPrescriptions`
**Detector**: StatusTransition (`status='expired'` 일괄 UPDATE 패턴, CC-005/DV-005/SB-005/IN-005 동일 형태 5번째)

## 트리거 조건
- **When**: 정기 batch (일/시간 단위, SYSTEM cron)
- **If**: `status='active'` AND `expires_at < now`
- **Then**: `UPDATE prescriptions SET status='expired'`, 마킹된 prescriptionIds 반환
- **Else**: 만료 안 됐을 시 제외

## 모니터링
- 일별 expired 마킹 카운트
- drug_code별 만료율 (재처방 패턴 분석)

## 신뢰도
- Detector: 75% (StatusTransition — batch UPDATE, CC-005/DV-005/SB-005/IN-005 입증된 동일 형태 5번째)
