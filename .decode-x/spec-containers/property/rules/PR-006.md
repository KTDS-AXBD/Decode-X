# PR-006: 명도 처리 Atomic

**Detector**: AtomicTransaction  
**Function**: `processEviction()`

## 규칙

명도 처리는 evictions 등록 + legal_proceedings + eviction_notifications + lease_closures + 임대 상태 종료를 원자적으로 수행한다.

## 트랜잭션 단계

1. INSERT evictions (명도 기록)
2. INSERT legal_proceedings (법적 절차)
3. INSERT eviction_notifications (임차인 통보)
4. INSERT lease_closures (계약 종료 이력)
5. UPDATE leases SET status='terminated'

## 근거

명도 처리 도중 부분 완료로 인한 법적 분쟁 방지.
