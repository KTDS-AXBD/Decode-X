# Runbook — CS-006 운영자 정산 atomic

## 알림 조건
`E404-RETURNED-RENTAL-SESSION` 빈도 급증 또는 atomic transaction rollback 발생 시.

## 진단
1. 반납된 session(`status='returned'`) 존재 여부
2. atomic 단계 실패 (operator_billing_records INSERT / operator_payouts INSERT / billing_records UPDATE)
3. billing_rate 계산 오류 또는 revenue 음수 입력 케이스 점검

## 해소
- 임시: 실패 session 수동 정산 (운영자 계좌 직접 입금 + 정합화)
- 영구: 일별 정산 자동 retry + billing_rate 단위 테스트 강화
