# Runbook — CN-001: 입찰 금액 한도 검증

## 트리거
시공사/입찰관리SYSTEM → `POST /api/v1/construction/projects/:id/bids`

## 처리 흐름
1. 프로젝트 status='bidding' 확인
2. `bidAmount >= MAX_BID_AMOUNT_LIMIT` (1,000,000,000 KRW) 차단
3. 검증 통과 시 bids INSERT

## 모니터링
- E422-BID-LIMIT 발생률 (한도 초과 입찰 비율)
- 입찰 금액 임계값 근접 비율 (>950,000,000 KRW)
