# Runbook — MR-001: 화물 적재 한도 검증

## 트리거
화주/화물관리SYSTEM → `POST /api/v1/maritime/vessels/:id/cargo`

## 처리 흐름
1. 선박 status='available' 또는 'loading' 확인
2. `cargoTons >= MAX_CARGO_CAPACITY_TONS` (200,000 tons) 차단
3. 선박 개별 적재 용량 초과 차단 (current_load + cargoTons > max_capacity)
4. 검증 통과 시 shipments INSERT + vessels.current_load_tons 갱신

## 모니터링
- E422-CARGO-LIMIT 발생률 (화물 한도 초과 비율)
- E422-VESSEL-CAPACITY 발생률 (선박 용량 초과 비율)
- 화물 톤수 임계값 근접 비율 (>180,000 tons)
