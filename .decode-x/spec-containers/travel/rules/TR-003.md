# TR-003 — 여정 확정 Atomic Transaction

**BL ID**: TR-003
**Domain**: Travel (여행)
**Detector**: AtomicTransaction
**Function**: `confirmItinerary()`

## Condition
여정 확정 처리 시 (itinerary.status='pending')

## Criteria
- `itinerary.status='pending'`
- 결제 금액 유효 (paymentAmount > 0)

## Outcome
Atomic: `itineraries.status='confirmed'` UPDATE + PNR 발급 + `trips` INSERT (status='pending')

## Exception
- `E404-IT`: 여정 미존재
- `E409-IT`: 상태 불일치 (pending 아님)
