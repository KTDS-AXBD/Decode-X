# Runbook — AR-001 갤러리 동시 active artwork 한도

## 알림 조건
`E422-GALLERY-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `galleries.active_artworks` vs `total_capacity` 비교 조회
2. 특정 갤러리/전시 시즌 동시 전시 작품 집중 여부 확인
3. `MAX_CONCURRENT_ACTIVE_ARTWORKS_PER_GALLERY` 기본값(60) 적절성 검토 (갤러리 평수 대비)

## 해소
- 임시: 갤러리에 archived 후보 artwork 정리 권유 또는 total_capacity 직접 조정
- 영구: 전시 회전율 정책 명문화 + 추가 갤러리 (sub-gallery) 신설 또는 순회 전시 도입
