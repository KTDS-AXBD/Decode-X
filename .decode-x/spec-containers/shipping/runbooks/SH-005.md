# Runbook — SH-005 만료 화물 일괄 마킹 결과 이상

## 알림 조건
`expireCargoLoadBatch` 결과 expiredCount가 평소 평균 대비 비정상 (급증/급감).

## 진단
1. 직전 batch run 시점 + 누락 batch 여부 확인 (cron 동작)
2. arrived 상태에서 정체된 화물 적체율 확인 (항만 수령 지연 트렌드)
3. `loaded_at <= now` 비교 시점 boundary 정확성 (timezone)

## 해소
- 임시: arrived 화물 항만 수령 안내 캠페인 (체선료 회피)
- 영구: arrived 정체 임계 도달 시 자동 알림 + 항만 별 수령 SLA 모니터링
