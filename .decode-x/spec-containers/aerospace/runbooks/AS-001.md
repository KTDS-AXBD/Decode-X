# Runbook — AS-001 항공우주 발사대 일일 발사 정원 한도

## 알림 조건
`E422-LAUNCH-PAD-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `launch_pad_pool.active_launches` vs `total_capacity` 비교 조회
2. 특정 발사장/시기(launch window) 발사 집중 여부 확인
3. `MAX_DAILY_LAUNCHES_PER_PAD` 기본값 적절성 검토 (launch window vs 평시)

## 해소
- 임시: 인접 발사장 발사 일정 안내 또는 total_capacity 직접 조정 (피크 흡수)
- 영구: 신규 발사대 증설 또는 launch window 분산 (다른 궤도/시간대 활용)
