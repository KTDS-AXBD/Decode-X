# WL-001 — 세션 정원 한도 검증

**함수**: `bookSession()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant fallback)

## 조건
세션 예약 요청 시 `session.booked_count >= capacity(또는 MAX_SESSION_CAPACITY)` 검사.

## 기준
`MAX_SESSION_CAPACITY = 20` (세션 정원 기본 한도, 인원)

## 결과
- 정원 내: 예약 허용, appointments 생성 + sessions.booked_count 증가
- 정원 초과: `E422-SESSION-CAPACITY-EXCEEDED` 예외

## 근거
웰니스/스파 서비스의 세션별 정원 관리 — 치료사 1인당 동시 시술 인원 제한.
