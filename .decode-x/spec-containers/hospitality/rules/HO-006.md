# HO-006 — 오버부킹 보상 트랜잭션

**BL ID**: HO-006
**Domain**: Hospitality (숙박)
**Detector**: AtomicTransaction
**Function**: `handleOverbookingCompensation()`

## Condition
오버부킹 감지 시 (availableRooms=0, booking.status='pending')

## Criteria
- `booking.status='pending'`
- 가용 객실 없음 (availableRooms=0)

## Outcome
Atomic: `overbooking_log` INSERT + `bookings.status='cancelled'` UPDATE + 보상 처리

## Exception
- `E404-BK`: 예약 미존재
- `E409-OB`: 이미 처리된 오버부킹
