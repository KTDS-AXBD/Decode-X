# ED-001: 학생 등록 연령 한도 검증

**Rule**: 6 ≤ studentAge ≤ 100 충족 시 학생 등록

**구현 함수**: `registerStudent`
**Detector**: ThresholdCheck (F445 Path A — UPPERCASE_CONSTANT)

## 트리거 조건
- **When**: 사용자 학생 등록 신청
- **If**: `MIN_STUDENT_AGE ≤ studentAge ≤ MAX_STUDENT_AGE`
- **Then**: `students` INSERT (status='active')
- **Else**: `E422-AGE-MIN` 또는 `E422-AGE-MAX`

## 신뢰도
- Detector: 70% (Path A)
