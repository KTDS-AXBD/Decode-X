# MR-002 — 운임 한도 비교

**BL ID**: MR-002
**Domain**: Maritime (해운)
**Detector**: ThresholdCheck (Path B — var-vs-var, `limit` keyword)
**Function**: `computeFreightRate()`

## Condition
운임 계산 시

## Criteria
- `quotedRate <= freightRateLimit` (var-vs-var, `freightRateLimit`에 `limit` keyword 포함)
- 항로 유형별 최대 운임 한도: intra_asia=15 / asia_europe=35 / transpacific=40 / transatlantic=45 (USD/ton)

## Outcome
운임 허용 + freight_rates INSERT

## Exception
- `E422-FREIGHT-RATE-EXCEEDED`: 운임 한도 초과 (항로 유형 기준)
- `E404-SHIPMENT`: 선적 미존재
