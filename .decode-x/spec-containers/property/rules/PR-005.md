# PR-005: 정기 점검 일괄 처리

**Detector**: StatusTransition (batch pattern)  
**Function**: `markInspectionBatch()`

## 규칙

`scheduledBefore` 이전 예정된 점검을 일괄로 'inspected' 상태로 전환한다.

## 조건

```
scheduled_date <= scheduledBefore AND status = 'scheduled' → UPDATE status='inspected'
```

## 근거

정기 점검 자동화로 관리 부담 경감. CC-005 batch 패턴 30번째 재사용.
