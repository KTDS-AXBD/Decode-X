# ED-004: 수강 상태 전환 (enrolled → completed/withdrawn/failed, completed → graded)

**Rule**: 허용된 상태 전환만 인정

**구현 함수**: `transitionEnrollmentStatus`
**Detector**: StatusTransition

## 트리거 조건
- **When**: 교수/학적팀 상태 변경
- **If**: 허용 매트릭스
  - `enrolled → completed` (정상 수료)
  - `enrolled → withdrawn` (취소)
  - `enrolled → failed` (불합격)
  - `completed → graded` (성적 입력 완료)
- **Then**: `UPDATE enrollments SET status=?, completed_at/withdrawn_at`
- **Else**: `E409-TR`

## 상태 머신
```
enrolled → completed → graded
enrolled → withdrawn / failed
```

## 신뢰도
- Detector: 75%
