# PR-003: 임대 계약 갱신 Atomic

**Detector**: AtomicTransaction  
**Function**: `renewLease()`

## 규칙

계약 갱신은 leases 갱신 + lease_renewals 이력 + deposits 정산을 원자적으로 수행한다.

## 트랜잭션 단계

1. UPDATE leases SET status='renewed', end_date, monthly_rent
2. INSERT lease_renewals (이력 기록)
3. INSERT deposits (보증금 조정)

## 근거

갱신 도중 실패 시 부분 상태 방지.
