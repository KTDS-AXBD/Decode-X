# MN-001 — 채광량 한도 검증

**함수**: `recordExtraction()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant)

## 조건
채광량 기록 요청 시 `totalExtracted >= MAX_EXTRACTION_QUOTA` 검사.

## 기준
`MAX_EXTRACTION_QUOTA = 50000` (일일 채광량 한도, 톤)

## 결과
- 한도 내: 채광 허용, extraction 생성
- 한도 초과: `E422-QUOTA-EXCEEDED` 예외

## 근거
광업 안전 규정 — 일일 채광량 과다로 인한 지반 불안정 방지.
