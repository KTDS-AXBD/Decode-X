# LG-001 — 화물 발송 한도 검증

**BL ID**: LG-001
**Domain**: Logistics (물류)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `checkShipmentLimits()`

## Condition
화물 발송 등록 요청 시

## Criteria
- `weightKg ≤ MAX_WEIGHT_KG` (30,000 kg)
- `volumeM3 ≤ MAX_VOLUME_M3` (100 m³)
- 무게 및 부피 양수 조건

## Outcome
`shipments` 테이블 INSERT (status='pending')

## Exception
- `E422-WT-MAX`: 무게 초과
- `E422-VL-MAX`: 부피 초과
- `E422-DIM-MIN`: 음수/0 값
