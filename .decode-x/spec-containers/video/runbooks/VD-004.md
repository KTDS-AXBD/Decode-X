# Runbook — VD-004 영상 상태 전환

## 알림 조건
`E409-PUBLISH` 빈도 급증 시 (잘못된 상태 전환 시도).

## 진단
1. 클라이언트 측 영상 상태 캐시 stale 여부 확인 (UI에서 retired/banned을 published처럼 표시)
2. 동시 transition 충돌 — 두 운영자가 동시에 같은 publish_id를 retired/banned로 전환
3. 콘텐츠 검수 워크플로 매트릭스 (encoded → published 자동 vs 검수 후 수동) 검토

## 해소
- 임시: 사용자 안내 + UI refresh 강제
- 영구: optimistic locking (version column) 도입 또는 transition matrix UI에 반영
