# Runbook — AR-004 artwork 상태 전환

## 알림 조건
`E409-SCHEDULE` 빈도 급증 시 (잘못된 상태 전환, 특히 긴급 withdrawn).

## 진단
1. 전시 시스템 측 상태 캐시 stale 여부 확인 (도록에서 withdrawn artwork를 exhibited처럼 표시)
2. 동시 transition 충돌 — 두 큐레이터가 동시에 같은 schedule_id를 withdraw/cancel 전환
3. 긴급 작품 회수 워크플로 매트릭스 검토 + 컬렉터 보상 정책 audit

## 해소
- 임시: 사용자 안내 + 도록 refresh 강제 + withdrawn notice 게시
- 영구: optimistic locking + withdrawn 시 컬렉터 보상 자동 스케줄
