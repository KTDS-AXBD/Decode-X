# IN-002: 청구 한도 검증

**Rule**: 청구 시 `coverage_limit - reserved_amount ≥ claim_amount` 검증

**구현 함수**: `checkClaimLimit`
**Detector**: ThresholdCheck (F445 Path B — `remainingCoverage < claimAmount` var-vs-var, `claimAmount` 변수의 `amount` keyword 매칭)

## 트리거 조건
- **When**: 청구 접수 직전 사전 검증
- **If**: `policy.status='active'` AND `coverage_limit - reserved_amount ≥ claim_amount`
- **Then**: `{canClaim: true, remainingCoverage}` 반환
- **Else**: `E404-PO` / `E409-ST` / `E422-LIMIT`

## 신뢰도
- Detector: 70% (ThresholdCheck Path B — `amount` keyword 매칭)
