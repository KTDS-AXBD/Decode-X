# TR-006 — 취소 환불 + 마일리지 복구 Atomic Transaction

**BL ID**: TR-006
**Domain**: Travel (여행)
**Detector**: AtomicTransaction
**Function**: `processCancellationRefund()`

## Condition
여정 취소 요청 시 (itinerary.status='pending' or 'confirmed')

## Criteria
- `itinerary.status IN ('pending', 'confirmed')`

## Outcome
Atomic: `itineraries.status='cancelled'` UPDATE + `refund_log` INSERT (환불 금액 + 마일리지 복구)

## Exception
- `E404-IT`: 여정 미존재
- `E409-IT`: 취소 불가 상태 (checked_in 이후)
