# LG-003 — 세관 통관 Atomic Transaction

**BL ID**: LG-003
**Domain**: Logistics (물류)
**Detector**: AtomicTransaction
**Function**: `clearCustoms()`

## Condition
화물 세관 통관 요청 시 (shipment.status='pending')

## Criteria
- `declaredValueUsd ≥ 0`

## Outcome
Atomic: `customs_records` INSERT (status='cleared') + `shipments.status='in_transit'`

## Exception
- `E404-SH`: 화물 미존재
- `E409-SH`: 상태 불일치
- `E422-VAL`: 신고 금액 음수
