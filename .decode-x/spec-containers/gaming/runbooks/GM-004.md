# Runbook — GM-004 게임 상태 전환 거부

## 알림 조건
`E409-LAUNCH` 빈도 급증 (잘못된 transition 시도).

## 진단
1. `previousStatus → newStatus` 매트릭스 위반 케이스 집계
2. UI/API 호출 측 잘못된 status 진입 여부 (race condition / 클라이언트 캐시)
3. 비정상 banned 비율 추적 (live → banned, published → banned)

## 해소
- 임시: 거부 케이스에 대한 정확한 안내 메시지 + 가능한 다음 status 후보 노출
- 영구: 클라이언트 측 status FSM 동기화 + observable transition 이벤트 발행
