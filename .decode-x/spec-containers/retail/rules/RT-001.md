# RT-001 — SKU 목록 가격 티어 한도 검증

**BL ID**: RT-001
**Domain**: Retail (소매)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `listSku()`

## Condition
SKU 목록 조회 요청 시

## Criteria
- `requestedTier ≤ MAX_SKU_PRICE_TIER` (최대 10 티어)
- `requestedTier > 0` (양수)

## Outcome
`sku_catalog` 테이블 조회 결과 반환

## Exception
- `E422-TIER-MAX`: 최대 가격 티어 초과
- `E422-TIER-MIN`: 가격 티어가 0 이하
