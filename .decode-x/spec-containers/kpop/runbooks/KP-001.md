# Runbook — KP-001 아레나 동시 active entry 한도

## 알림 조건
`E422-ARENA-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `arenas.active_entries` vs `total_capacity` 비교 조회
2. 특정 아레나/티켓오픈일/콘서트 회차(스타급 아티스트) 동시 예매 집중 여부 확인
3. `MAX_CONCURRENT_ACTIVE_ENTRIES_PER_ARENA` 기본값(50000) 적절성 검토 (소방법 안전 수용 인원/좌석 배치 기준)

## 해소
- 임시: 아레나에 ended 후보 entry 정리 권유 또는 total_capacity 직접 조정
- 영구: 추가 회차/지방 투어 일정 신설 + 스탠딩 구역 안전 수용 인원 재산정 + 인터파크/멜론티켓 대기열 분산
