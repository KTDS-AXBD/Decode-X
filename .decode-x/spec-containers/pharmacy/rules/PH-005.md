# PH-005 — 리콜 대상 약품 일괄 통지

**BL ID**: PH-005
**Domain**: Pharmacy (제약/약국)
**Detector**: StatusTransition (batch 패턴 — CC-005 batch 18번째 재사용)
**Function**: `markRecalledBatches()`

## Condition
리콜 배치 처리 시 (정기 스케줄러 또는 긴급 리콜 발동)

## Criteria
- `drugs.status = 'active'`
- `drugs.recalled_at IS NOT NULL AND recalled_at <= recallCutoffDate`

## Outcome
`drugs.status='recalled'` 일괄 UPDATE

## Exception
- 대상 없으면 markedCount=0 (정상 처리)
