# MD-006 — 콘텐츠 테이크다운 atomic

**BL ID**: MD-006
**Domain**: Media (미디어)
**Detector**: AtomicTransaction
**Function**: `processTakedown()`

## Condition
콘텐츠 테이크다운 처리 시 (신고 → 검토 → 비공개 → 환불)

## Criteria
- `content.status != 'archived'` AND `content.status != 'expired'`
- `report` 존재 확인

## Outcome
atomic: `contents.status='archived'` + `reports.status='resolved'` + `licenses.status='revoked'` + `refunds` INSERT (활성 라이선스 건수만큼)

## Exception
- `E404-CONTENT`: 콘텐츠 미존재
- `E404-REPORT`: 신고 미존재
- `E409-CONTENT`: 이미 비공개/만료된 콘텐츠
