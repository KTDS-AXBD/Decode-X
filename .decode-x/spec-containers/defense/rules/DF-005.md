# DF-005 — 훈련 교체 일괄 처리

**함수**: `markTrainingRotation()`
**Detector**: StatusTransition (batch — CC-005 25번째 재사용)

## 조건
훈련 일괄 처리 요청 시 `scheduled_at <= scheduledBefore` AND `status = 'scheduled'` 조건 만족.

## 기준
`training_schedules.status = 'scheduled'` → `'completed'`

## 결과
- 조건 만족: `status='completed', completed_at=NOW()` 일괄 UPDATE
- 대상 없음: completedCount=0 반환

## 근거
부대 훈련 관리 — 훈련 일정 일괄 완료 처리로 훈련 이력 정확도 유지.
