# BT-001 — 좌석 정원 한도 검증

**함수**: `bookSeat()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant fallback)

## 조건
좌석 예약 요청 시 `seat.booked_count >= capacity(또는 MAX_SEAT_CAPACITY)` 검사.

## 기준
`MAX_SEAT_CAPACITY = 20` (좌석 정원 기본 한도, 인원)

## 결과
- 정원 내: 예약 허용, seat_bookings 생성 + beauty_seats.booked_count 증가
- 정원 초과: `E422-SEAT-CAPACITY-EXCEEDED` 예외

## 근거
미용실의 좌석별 정원 관리 — 스타일리스트 1인당 동시 서비스 가능 고객 제한.
