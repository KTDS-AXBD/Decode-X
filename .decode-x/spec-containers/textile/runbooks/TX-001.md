# Runbook — TX-001 방직 공장 동시 직조 batch 한도

## 알림 조건
`E422-MILL-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `mills.active_batches` vs `total_capacity` 비교 조회
2. 특정 공장/제품군/시즌(F/W, S/S) 동시 직조 집중 여부 확인
3. `MAX_CONCURRENT_BATCHES_PER_MILL` 기본값(300 bolt) 적절성 검토 (수출 vs 내수)

## 해소
- 임시: 구매자에게 다른 공장 안내 또는 total_capacity 직접 조정 (peak 흡수)
- 영구: 공장 capacity 증설 또는 시즌 분산 (R&D 시즌 인센티브)
