# MF-002 — 생산 주문 용량 한도 검증

**BL ID**: MF-002
**Domain**: Manufacturing (제조)
**Detector**: ThresholdCheck (Path B — var-vs-var, `capacityLimit` keyword 매칭)
**Function**: `placeProductionOrder()`

## Condition
생산 주문 요청 시

## Criteria
- `requiredCapacity ≤ capacityLimit` (가용 용량 내 주문)
- `requiredCapacity > 0` (양수 조건)

## Outcome
`production_orders` 테이블 INSERT (status='pending')

## Exception
- `E422-CAP-MAX`: 최대 생산 용량 초과
- `E422-CAP-MIN`: 0 이하 용량 요청
