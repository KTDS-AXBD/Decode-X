# HC-001: 환자 등록 연령 한도 검증

**Rule**: 0 ≤ patientAge ≤ 130 충족 시 환자 등록, 미충족 시 거절

**구현 함수**: `registerPatient`
**Detector**: ThresholdCheck (F445 Path A — UPPERCASE_CONSTANT 비교)

## 트리거 조건
- **When**: 사용자 환자 등록 신청
- **If**: `MIN_PATIENT_AGE ≤ patientAge ≤ MAX_PATIENT_AGE`
- **Then**: `patients` INSERT (status='active')
- **Else**: `E422-AGE-MIN` 또는 `E422-AGE-MAX`

## 신뢰도
- Detector: 70% (Path A — UPPERCASE_CONSTANT 비교)
