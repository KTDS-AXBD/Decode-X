# BK-005 — 휴면 계좌 일괄 동결 처리

**BL ID**: BK-005
**Domain**: Banking (은행)
**Detector**: StatusTransition (batch 패턴 — CC-005 batch 16번째 재사용)
**Function**: `markDormantAccounts()`

## Condition
휴면 계좌 일괄 처리 (배치)

## Criteria
- `accounts.status = 'active'`
- 최근 거래 날짜 < `inactiveCutoffDate` (장기 미거래 기준일)

## Outcome
`accounts.status='dormant'` 일괄 UPDATE + `dormant_at` 기록

## Exception
- 대상 없으면 `markedCount=0` (정상 완료)
