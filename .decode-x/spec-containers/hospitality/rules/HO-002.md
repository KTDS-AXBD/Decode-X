# HO-002 — 취소 환불 정책 (cutoff 시간 비교)

**BL ID**: HO-002
**Domain**: Hospitality (숙박)
**Detector**: ThresholdCheck (Path B — var-vs-var, `cancellationLimitHours` 'limit' keyword 매칭)
**Function**: `applyCancellationPolicy()`

## Condition
예약 취소 요청 시 (booking.status='pending' or 'confirmed')

## Criteria
- `hoursUntilCheckIn > cancellationLimitHours` (체크인 24시간 전까지 취소 허용)

## Outcome
전액 환불 처리 (refundAmount = 결제 금액 전액)

## Exception
- `E404-BK`: 예약 미존재
- `E409-BK`: 이미 체크인 또는 취소된 예약
- `E422-CANCEL-EXP`: 취소 기간 초과 (체크인 24시간 이내)
