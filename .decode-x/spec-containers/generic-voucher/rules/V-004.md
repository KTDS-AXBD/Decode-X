# V-004: 자동 소멸 (잔액 ≤ 1,000원)

**Rule**: 잔액 ≤ 1,000원 + status=active 시 자동 destroy, 미달 시 active 유지

| 조건 | 조건충족 | 결과 | 예외 |
|------|---------|------|------|
| balance ≤ 1000 + active | (자동) | status=destroyed, balance=0 | balance > 1000 시 active 유지 |

**구현 함수**: `autoDestroyVoucher`
**Detector**: StatusTransition (`active → destroyed`)
