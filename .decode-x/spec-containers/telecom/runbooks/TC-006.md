# Runbook — TC-006: 번호이동 atomic 트랜잭션

## 트리거
번호이동SYSTEM → `POST /api/v1/telecom/subscriptions/:id/port-out`

## 처리 흐름
1. 회선 조회 + status != 'terminated' 검증
2. atomic tx:
   - `port_out_requests` INSERT (status='completed')
   - `subscriptions.status='terminated'` UPDATE
   - `terminated_at` 기록

## 모니터링
- 번호이동 성공율
- E409-SUBSCRIPTION (이미 해지된 회선 시도)
- 정산 금액 합계 (carrier 정산 검증)
