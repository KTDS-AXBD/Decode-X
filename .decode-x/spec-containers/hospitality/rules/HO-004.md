# HO-004 — 예약 상태 전환 (pending → confirmed → checked_in → checked_out)

**BL ID**: HO-004
**Domain**: Hospitality (숙박)
**Detector**: StatusTransition
**Function**: `transitionBookingStatus()`

## Condition
예약 상태 변경 요청 시

## Criteria
허용 전환 매트릭스:
- `pending` → `confirmed`
- `confirmed` → `checked_in`
- `checked_in` → `checked_out`

## Outcome
`bookings.status` UPDATE

## Exception
- `E404-BK`: 예약 미존재
- `E409-TR`: 허용되지 않은 상태 전환
