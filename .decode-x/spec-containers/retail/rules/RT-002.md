# RT-002 — 프로모션 자격 최소 주문액 한도 비교

**BL ID**: RT-002
**Domain**: Retail (소매)
**Detector**: ThresholdCheck (Path B — var-vs-var, `minOrderLimit` keyword 매칭)
**Function**: `applyPromotion()`

## Condition
프로모션 적용 요청 시

## Criteria
- `cartTotal ≥ minOrderLimit` (프로모션별 최소 주문액 이상)
- `promotion.status = 'active'`

## Outcome
할인 금액(10%) 계산 + `eligibleAmount` 반환

## Exception
- `E422-PROMO-MIN`: 장바구니 금액이 최소 주문액 미달
- `E404-PROMO`: 프로모션 미존재
- `E409-PROMO`: 프로모션 비활성 상태
