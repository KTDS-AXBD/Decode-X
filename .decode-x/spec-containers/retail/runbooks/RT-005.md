# Runbook — RT-005: 재고 일괄 동기화 배치

## 트리거
재고 시스템 배치 → `POST /api/v1/retail/inventory/sync` (cron)

## 처리 흐름
1. stock_status='available' SKU 후보 조회 (product_id 범위)
2. 각 SKU stock_status='synced' 업데이트
3. inventory_sync_log INSERT (감사 로그)

## 모니터링
- syncedCount 추이 (0이면 재고 동기화 불필요 또는 오류)
- 배치 소요 시간
- 시간당 동기화 건수
