# Runbook — AQ-001 수족관 동시 active admit 한도

## 알림 조건
`E422-AQUARIUM-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `aquariums.active_admits` vs `total_capacity` 비교 조회
2. 특정 수족관/성수기(주말/방학/우천일) 동시 입장 집중 여부 확인
3. `MAX_CONCURRENT_ACTIVE_ADMITS_PER_AQUARIUM` 기본값(8000) 적절성 검토 (소방법 안전 수용 인원/수조 관람 동선 기준)

## 해소
- 임시: 수족관에 ended 후보 admit 정리 권유 또는 total_capacity 직접 조정
- 영구: 시간대 분산 입장권 도입 + 야간 관람 신설 + 동선 분산 (특별전 별도 입장)
