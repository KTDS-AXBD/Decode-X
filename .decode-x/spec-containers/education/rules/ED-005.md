# ED-005: 학기 만료 시 미수료 자동 마킹

**Rule**: term.term_end < now AND enrollment.status='enrolled' → status='failed' 자동 마킹

**구현 함수**: `markFailedEnrollmentsByTerm`
**Detector**: StatusTransition (`status='failed'` 일괄 UPDATE 패턴, CC/DV/SB/IN/HC-005 동일 형태 6번째)

## 트리거 조건
- **When**: 학기 종료 직후 batch (SYSTEM cron)
- **If**: `term.term_end < now` AND enrollment status='enrolled'
- **Then**: `UPDATE enrollments SET status='failed', completed_at=now`, 마킹된 enrollmentIds 반환
- **Else**: term 미존재 또는 term_end 미도달 시 skip

## 모니터링
- 학기별 failed 마킹 카운트
- 과목별 미수료율 (커리큘럼 입력)

## 신뢰도
- Detector: 75% (StatusTransition — batch UPDATE, CC/DV/SB/IN/HC-005 입증된 동일 형태 6번째)
