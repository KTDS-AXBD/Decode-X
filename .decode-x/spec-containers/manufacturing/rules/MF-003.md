# MF-003 — 생산 주문 확정 Atomic 트랜잭션

**BL ID**: MF-003
**Domain**: Manufacturing (제조)
**Detector**: AtomicTransaction
**Function**: `confirmProductionOrder()`

## Condition
생산 주문 확정 처리 시

## Criteria
- `order.status = 'pending'` (확정 대기 상태)

## Outcome
atomic: `production_orders.status = 'confirmed'` + `production_lots` INSERT (status='planned')

## Exception
- `E404-ORD`: 생산 주문 미존재
- `E409-ORD`: pending 외 상태에서 확정 시도
