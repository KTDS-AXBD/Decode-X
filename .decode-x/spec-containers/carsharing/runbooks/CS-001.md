# Runbook — CS-001 카쉐어링 fleet 차량 정원 한도

## 알림 조건
`E422-FLEET-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `vehicle_pool.active_vehicles` vs `total_vehicles` 비교 조회
2. 특정 풀/시간대 예약 집중 여부 확인
3. `MAX_FLEET_VEHICLES` 기본값 적절성 검토 (피크 시간 vs 평시)

## 해소
- 임시: 인접 풀 차량 재배치 또는 total_vehicles 직접 조정
- 영구: 신규 풀 개설 또는 시간대 분산 요금제 도입 (출퇴근 시간대 가격 인상)
