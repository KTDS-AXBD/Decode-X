# Runbook — AG-006: 작물 인증 atomic

## 트리거
인증기관/인증관리SYSTEM → `POST /api/v1/agriculture/crops/:id/certify`

## 처리 흐름
1. crop.status = 'harvested' 또는 'sold' 확인
2. grade != 'rejected' 확인 (등급 미달 차단)
3. atomic transaction: certifications INSERT + certification_labels INSERT + crops.certified=1
4. 만료일 자동 계산 (발급일 + 1년)

## 모니터링
- E422-CROP-REJECTED 발생률 (등급 미달 인증 시도)
- 인증 유형별 발급 건수 (organic/gmo-free/fair-trade)
- 인증 만료 예정 건수 (30일 이내)
