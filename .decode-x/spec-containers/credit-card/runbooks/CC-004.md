# Runbook — CC-004: 카드 상태 전환

## 트리거
- 관리자 차단 (사고 신고) → blocked
- 만료 batch (`expires_at < now`) → expired
- 사용자 해지 요청 → cancelled

## 처리 흐름
1. `SELECT status FROM credit_cards WHERE id=?`
2. `status != 'active'` → `E409-TR` (전환 불가)
3. `UPDATE credit_cards SET status = ?`
4. `{previousStatus, newStatus}` 반환 (audit log 입력)

## 알림
- blocked 전환 시 사용자 SMS/email 즉시 발송
- delinquent 전환은 CC-005 batch 별도

## 모니터링
- 일별 status 전환 카운트 (per status)
