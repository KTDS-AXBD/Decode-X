# MF-005 — 불량 Lot 일괄 격리 배치

**BL ID**: MF-005
**Domain**: Manufacturing (제조)
**Detector**: StatusTransition (batch 패턴 — CC/DV/SB/IN/HC/ED/RE/LG/HO/TR-005 11번째 재사용)
**Function**: `quarantineDefectiveLots()`

## Condition
불량 lot 일괄 격리 처리 시 (시스템 배치)

## Criteria
- `production_lots.status IN ('in_progress', 'qc')` (격리 대상 상태)
- 해당 주문 ID에 속하는 lot

## Outcome
`production_lots.status = 'quarantined'` 일괄 UPDATE + `defect_quarantine_log` INSERT

## Exception
대상 lot 없으면 `quarantinedCount = 0` (정상 처리)
