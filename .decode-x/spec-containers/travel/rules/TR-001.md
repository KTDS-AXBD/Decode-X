# TR-001 — 항공 예약 좌석 가용성 검증

**BL ID**: TR-001
**Domain**: Travel (여행)
**Detector**: ThresholdCheck (Path A — var-vs-UPPERCASE)
**Function**: `bookFlight()`

## Condition
항공 예약 요청 시

## Criteria
- `seatsRequested ≤ availableSeats` (가용 좌석 이내)
- `seatsRequested ≤ MAX_SEATS_PER_BOOKING` (1회 최대 9석)
- `seatsRequested > 0` (양수 조건)

## Outcome
`itineraries` 테이블 INSERT (status='pending')

## Exception
- `E422-ST-MAX`: 최대 예약 좌석 초과
- `E422-ST-AVAIL`: 가용 좌석 부족
- `E422-ST-MIN`: 0 이하 좌석 수
