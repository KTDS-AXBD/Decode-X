# Runbook — AM-004 ticket 상태 전환

`E409-SCHEDULE` 빈도 급증 시 (잘못된 상태 전환, 특히 긴급 revoked).

## 진단

3. 어트랙션 운영중단/안전 워크플로 매트릭스 검토 + 방문객 보상 정책 audit

## 해소
- 임시: 사용자 안내 + 앱 refresh 강제 + revoked notice 송출
- 영구: optimistic locking + revoked 시 방문객 보상 자동 스케줄 + 어트랙션 대체 추천 hook
