# RT-005 — 재고 일괄 동기화 배치

**BL ID**: RT-005
**Domain**: Retail (소매)
**Detector**: StatusTransition (batch 패턴 — CC-005 12번째 재사용)
**Function**: `markInventorySync()`

## Condition
재고 동기화 배치 처리 시 (시스템 자동)

## Criteria
`sku_catalog.stock_status = 'available'` AND 동기화 대상 SKU 확인

## Outcome
`sku_catalog.stock_status='synced'` 일괄 UPDATE + `inventory_sync_log` INSERT

## Exception
대상 없으면 syncedCount=0 (정상 처리, 예외 없음)
