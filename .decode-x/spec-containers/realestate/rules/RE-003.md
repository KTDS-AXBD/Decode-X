# RE-003: 계약 체결 atomic transaction

**Rule**: property listed 검증 후 atomic하게 lease INSERT + property status='leased'

**구현 함수**: `signLease`
**Detector**: AtomicTransaction (`db.transaction(() => { ... }); tx();`)

## 트리거 조건
- **When**: 양 당사자 계약 합의
- **If**: `property.status='listed'`
- **Then**: atomic transaction
  1. `INSERT INTO leases (status='active', start_at, end_at)`
  2. `UPDATE properties SET status='leased'`
- **Else**: `E404-PR` / `E409-PR`

## 무결성 보장
- 두 변경 atomic — 중복 임차 방지

## 신뢰도
- Detector: 85%
