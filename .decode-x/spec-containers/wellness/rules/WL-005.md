# WL-005 — 노쇼 세션 일괄 처리

**함수**: `markNoShowSessions()`
**Detector**: StatusTransition (batch 패턴, CC-005 28번째 재사용)

## 조건
배치 실행 시 `sessions.scheduled_date <= scheduledBefore AND appointments.status = 'confirmed'`.

## 기준
기준일 이전 미참석 예약(confirmed 상태)을 no_show로 일괄 처리.

## 결과
- 대상 있음: `appointments.status = 'no_show'` 일괄 UPDATE
- 대상 없음: markedCount = 0

## 근거
예약 후 미참석 회원 노쇼 정책 자동화. 취소 수수료 부과 및 블랙리스트 관리를 위한 선행 단계.
