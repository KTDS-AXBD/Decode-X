# Runbook — MU-001 음악 스트리밍 티어 동시 세션 정원 한도

## 알림 조건
`E422-STREAMING-TIER-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `streaming_tiers.active_sessions` vs `total_capacity` 비교 조회
2. 특정 티어/시간대(peak hour) 동시 세션 집중 여부 확인
3. `MAX_CONCURRENT_SESSIONS_PER_TIER` 기본값 적절성 검토 (peak vs 평시)

## 해소
- 임시: 사용자에게 다른 티어 안내 또는 total_capacity 직접 조정 (피크 흡수)
- 영구: 인프라 capacity 증설 또는 peak hour 분산 (오프 피크 인센티브)
