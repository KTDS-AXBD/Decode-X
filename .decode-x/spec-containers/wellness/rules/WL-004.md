# WL-004 — 예약 상태 전환

**함수**: `transitionAppointmentStatus()`
**Detector**: StatusTransition

## 조건
예약 상태 전환 요청 시 허용 매트릭스 검증.

## 기준
허용 전환:
- `booked → confirmed`
- `confirmed → in_session`
- `in_session → completed`
- `completed → reviewed`

## 결과
- 허용: appointments.status UPDATE
- 불허: `E409-APPOINTMENT` (invalid transition)

## 근거
웰니스 세션 예약 lifecycle 제어. 순서를 건너뛴 상태 변경 방지 (예: booked → completed 직접 전환 불가).
