# Runbook — TH-001 극장 동시 active seat 한도

## 알림 조건
`E422-THEATER-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `theaters.active_seats` vs `total_capacity` 비교 조회
2. 특정 극장/관/시간대(주말 황금 시간대) 동시 예매 집중 여부 확인
3. `MAX_CONCURRENT_ACTIVE_SEATS_PER_THEATER` 기본값(2000) 적절성 검토 (관별 좌석 수 대비)

## 해소
- 임시: 극장에 ended 후보 seat 정리 권유 또는 total_capacity 직접 조정
- 영구: 회차 추가 신설 + 멀티플렉스 관 증설 + 좌석 등급 분산 정책 명문화
