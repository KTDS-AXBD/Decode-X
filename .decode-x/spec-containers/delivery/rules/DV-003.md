# DV-003: 배송 시작 atomic transaction

**Rule**: PENDING 상태 검증 후 atomic하게 status='SHIPPED' + delivery_events INSERT

**구현 함수**: `startShipping`
**Detector**: AtomicTransaction (`db.transaction(() => { ... }); tx();` 패턴)

## 트리거 조건
- **When**: 배송 출고 처리 트리거
- **If**: `delivery.status = 'PENDING'`
- **Then**: atomic transaction
  1. `UPDATE deliveries SET status='SHIPPED', shipped_at=now`
  2. `INSERT INTO delivery_events (event_type='SHIPPED')`
- **Else**: `E404-DV` 또는 `E409-ST`

## 무결성 보장
- 두 UPDATE/INSERT atomic — 실패 시 자동 rollback
- delivery_events SHIPPED 누락 시 audit 추적 불가 → atomic 필수

## 신뢰도
- Detector: 85% (AtomicTransaction — better-sqlite3 표준)
