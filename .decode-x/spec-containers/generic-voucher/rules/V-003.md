# V-003: 사용 시 잔액 차감 (atomic)

**Rule**: balance update + ledger insert를 atomic transaction으로 처리, 무결성 실패 시 rollback

| 조건 | 조건충족 | 결과 | 예외 |
|------|---------|------|------|
| 사용 + 잔액 충분 | `balance >= amount` | UPDATE balance + INSERT ledger (atomic) | rollback on integrity fail |

**구현 함수**: `redeemVoucher`
**Detector**: AtomicTransaction (`db.transaction()(() => { ... })`)
