# CH-001 — 기부 영수증 발급 한도 검증

**함수**: `recordDonation()`
**Detector**: ThresholdCheck (Path A, UPPERCASE constant)

## 조건
기부 영수증 발급 요청 시 `amount >= MAX_RECEIPT_AMOUNT` 검사.

## 기준
`MAX_RECEIPT_AMOUNT = 10_000_000` (영수증 발급 최대 한도, KRW 1천만)

## 결과
- 한도 내: 영수증 발급 허용, donation_receipts 생성
- 한도 초과: `E422-RECEIPT-LIMIT-EXCEEDED` 예외

## 근거
비영리단체 기부금 영수증 발급 규정 — 한도 초과 기부는 별도 심사 필요.
