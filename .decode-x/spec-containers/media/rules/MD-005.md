# MD-005 — 만료 콘텐츠 일괄 처리

**BL ID**: MD-005
**Domain**: Media (미디어)
**Detector**: StatusTransition (batch 패턴 — CC-005 batch 17번째 재사용)
**Function**: `markExpiringContent()`

## Condition
만료 콘텐츠 일괄 처리 (배치)

## Criteria
- `contents.status = 'published'`
- `expires_at IS NOT NULL`
- `expires_at <= expirationCutoffDate`

## Outcome
`contents.status='expired'` 일괄 UPDATE (대상 건 전체)

## Exception
- 대상 없으면 markedCount=0 (정상 종료)
