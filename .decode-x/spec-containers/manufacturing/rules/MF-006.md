# MF-006 — 출하 해제 Atomic 트랜잭션

**BL ID**: MF-006
**Domain**: Manufacturing (제조)
**Detector**: AtomicTransaction
**Function**: `releaseForShipment()`

## Condition
QC 통과 lot 출하 해제 시

## Criteria
- `lot.status = 'qc'` (QC 완료 상태)

## Outcome
atomic: `production_lots.status = 'released'` + `shipment_log` INSERT

## Exception
- `E404-LOT`: 생산 lot 미존재
- `E409-LOT`: qc 외 상태에서 출하 해제 시도
