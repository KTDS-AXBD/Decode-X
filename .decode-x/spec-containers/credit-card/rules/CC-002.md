# CC-002: 결제 한도 검증

**Rule**: 결제 승인 전 (`used_amount + amount`) ≤ `credit_limit` 검증, 한도 초과 시 거절

**구현 함수**: `checkPaymentLimit`
**Detector**: ThresholdCheck (`remainingLimit < amount`)

## 트리거 조건
- **When**: 결제 승인 직전 (CC-003 호출 전)
- **If**: `card.status = 'active'` AND `credit_limit - used_amount ≥ amount`
- **Then**: `{canPay: true, remainingLimit}` 반환
- **Else**: `E404-CARD` (미존재) / `E409-ST` (비활성) / `E422-LIMIT` (한도 초과)

## 신뢰도
- Detector: 70% (ThresholdCheck — `remainingLimit < amount` 명시 비교)
