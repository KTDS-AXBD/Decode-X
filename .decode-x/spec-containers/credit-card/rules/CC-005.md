# CC-005: 연체 자동 마킹

**Rule**: 사용금액 > 0 AND 마지막 결제일 + 30일 초과 시 status → 'delinquent' 자동 마킹

**구현 함수**: `markDelinquentCards`
**Detector**: StatusTransition (`status='delinquent'` 일괄 UPDATE 패턴)

## 트리거 조건
- **When**: 정기 batch (일/시간 단위, SYSTEM)
- **If**: `status='active'` AND `used_amount > 0` AND `last_payment_at < now - 30days`
- **Then**: `UPDATE credit_cards SET status='delinquent', delinquent_since=now`, 마킹된 cardIds 반환
- **Else**: 결제 이력 있거나 미사용 시 marking 제외

## 모니터링
- 일별 delinquent 신규 마킹 카운트 (대시보드)
- delinquent_since 분포 (신용 정책 입력)

## 신뢰도
- Detector: 75% (StatusTransition — batch UPDATE 패턴)
