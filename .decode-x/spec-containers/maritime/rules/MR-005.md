# MR-005 — 항구 처리 일괄 갱신

**BL ID**: MR-005
**Domain**: Maritime (해운)
**Detector**: StatusTransition (batch 패턴 — CC-005 batch 21번째 재사용)
**Function**: `markPortHandled()`

## Condition
항구 처리 일괄 갱신 (배치)

## Criteria
- shipments.status = 'arrived'
- arrived_at <= arrivalCutoff
- port_handled = 0

## Outcome
port_handled=1, port_handled_at 기록, status='port_cleared' 일괄 UPDATE

## Exception
- 대상 없으면 handledCount=0 (정상 종료)
