# HC-004: 예약 상태 전환 (booked → checked_in/cancelled/no_show, checked_in → completed/cancelled)

**Rule**: 허용된 상태 전환만 인정

**구현 함수**: `transitionAppointmentStatus`
**Detector**: StatusTransition

## 트리거 조건
- **When**: 의료진 또는 사용자 상태 변경
- **If**: 허용 매트릭스 충족
  - `booked → checked_in` (도착 체크인)
  - `booked → cancelled` (사전 취소)
  - `booked → no_show` (미출석)
  - `checked_in → completed` (정상 진료 완료)
  - `checked_in → cancelled` (도착 후 취소)
- **Then**: `UPDATE appointments SET status=?, cancelled_at=?` (cancelled 시점만)
- **Else**: `E409-TR`

## 상태 머신
```
booked → checked_in / cancelled / no_show
checked_in → completed / cancelled
```

## 신뢰도
- Detector: 75% (StatusTransition)
