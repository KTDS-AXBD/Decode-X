# SB-001: 구독 생성 가격/주기 한도 검증

**Rule**: priceKrw ≥ 1,000 AND 7 ≤ cycleDays ≤ 365 충족 시 구독 생성, 미충족 시 거절

**구현 함수**: `createSubscription`
**Detector**: ThresholdCheck (F445 Path A — `priceKrw < MIN_PRICE_KRW` / `cycleDays < MIN_CYCLE_DAYS` / `cycleDays > MAX_CYCLE_DAYS` UPPERCASE)

## 트리거 조건
- **When**: 사용자 신규 구독 등록
- **If**: `priceKrw ≥ MIN_PRICE_KRW` AND `MIN_CYCLE_DAYS ≤ cycleDays ≤ MAX_CYCLE_DAYS`
- **Then**: `subscriptions` INSERT (status='active', cycle_count=0, next_charge_at = now + cycleDays)
- **Else**: `E422-PR` / `E422-CY-MIN` / `E422-CY-MAX`

## 신뢰도
- Detector: 70% (ThresholdCheck Path A — UPPERCASE_CONSTANT 비교)
