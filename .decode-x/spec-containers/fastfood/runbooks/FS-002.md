# Runbook — FS-002 회원 콤보 할인 한도

## 알림 조건
`E422-COMBO-DISCOUNT-LIMIT-EXCEEDED` 빈도 급증 시.

## 진단
1. `customer_memberships.discount_used` vs `discount_limit` 비교 조회
2. 특정 tier_code 회원 비중 확인 (basic이 가장 자주 차단되는지)
3. 만료 임박 회원 잔여 할인 한도 분포 조회

## 해소
- 임시: 회원 일대일 협의로 한도 추가 또는 상위 tier 업그레이드 권유
- 영구: tier별 discount_limit 정책 재설계 (basic 5,000원 / silver 15,000원 / gold 무제한 등)
