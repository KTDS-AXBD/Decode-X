# LG-005 — 재고 Stale 자동 처리 (Batch)

**BL ID**: LG-005
**Domain**: Logistics (물류)
**Detector**: StatusTransition (batch — CC/DV/SB/IN/HC/ED/RE-005 동일 패턴 8번째)
**Function**: `markStaleInventory()`

## Condition
정기 batch 실행 시

## Criteria
- `status='active'` AND `last_updated < cutoffDate`

## Outcome
해당 재고 `status='stale'` 일괄 UPDATE, 변경된 inventoryIds 반환

## Exception
- cutoff 미도달 재고는 skip (정상 동작)
