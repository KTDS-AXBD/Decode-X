# Runbook — P-002: 연간 적립 한도 검증

**Rule**: (기적립 + 요청액) ≤ 1,800만원이면 적립 승인

## 트리거
- `POST /api/v1/pension/{accountId}/accumulate`

## 전제조건
- `amount`: 이번 적립 요청 금액 (원 단위, 양수)
- `accumulatedThisYear`: 역년(1/1~12/31) 기준 기적립 합계 (D1 조회)
- `ANNUAL_LIMIT_KRW = 18_000_000`

## 처리 흐름
1. `amount <= 0` → `E422-INVALID_AMOUNT` 반환
2. `accumulatedThisYear + amount > ANNUAL_LIMIT_KRW` → `E422-LIMIT_EXCEEDED` 반환
3. 조건 통과 시 → `pension_ledger` INSERT + balance 갱신 + `PensionAccumulated` 이벤트
