# Runbook — PB-001 임프린트 볼륨 동시 등록 한도

## 알림 조건
`E422-IMPRINT-VOLUME-EXCEEDED` 빈도 급증 시.

## 진단
1. `imprints.active_volumes` vs `total_capacity` 비교 조회
2. 특정 임프린트/장르/시즌(신학기, 연말) 동시 등록 집중 여부 확인
3. `MAX_CONCURRENT_VOLUMES_PER_IMPRINT` 기본값(500) 적절성 검토 (피크 vs 평시)

## 해소
- 임시: 저자에게 다른 임프린트 안내 또는 total_capacity 직접 조정 (피크 흡수)
- 영구: 임프린트 capacity 증설 또는 시즌 분산 (오프 시즌 인센티브)
