# MD-002 — 무료 시청 한도 비교

**BL ID**: MD-002
**Domain**: Media (미디어)
**Detector**: ThresholdCheck (Path B — var-vs-var, viewQuotaLimit keyword 매칭)
**Function**: `checkViewQuota()`

## Condition
콘텐츠 시청 시

## Criteria
- `viewedCount <= viewQuotaLimit` (tier별 월간 한도: free=10, basic=50, standard=200, premium=999)
- `viewQuotaLimit` 왼쪽 변수에 `limit` keyword 포함 — F445 Path B 패턴

## Outcome
시청 허용 + 카운트 증가

## Exception
- `E422-VIEW-QUOTA`: 무료 시청 한도 초과
