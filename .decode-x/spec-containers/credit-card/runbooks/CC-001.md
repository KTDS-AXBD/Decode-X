# Runbook — CC-001: 카드 발급 신용평가

## 트리거
`POST /api/v1/credit-cards/issue` 또는 발급 신청 batch

## 처리 흐름
1. 신용점수 + 연소득 검증 (`creditScore ≥ 600` AND `annualIncome ≥ 24M`)
2. 한도 자동 산정 (`baseLimit = annualIncome × 0.3` + scoreBonus)
3. `credit_cards` INSERT (status='active', expires_at = now + 5y)
4. PAN 발급 + 카드 발송 트리거 (외부)

## 모니터링
- 발급 거절률 (E422-CS / E422-IN 비율)
- 평균 approvedLimit 분포
