# PH-002 — 처방 잔여 횟수 한도 비교

**BL ID**: PH-002
**Domain**: Pharmacy (제약/약국)
**Detector**: ThresholdCheck (Path B — var-vs-var, `refillQuotaLimit` keyword)
**Function**: `checkRefillQuota()`

## Condition
처방 리필 요청 시

## Criteria
- `refillsUsed <= refillQuotaLimit` (처방 유형별 한도)
- antibiotic: 1회 / painkiller: 3회 / chronic: 12회 / supplement: 6회

## Outcome
리필 허용 + canRefill=true

## Exception
- `E422-REFILL-QUOTA`: 처방 잔여 횟수 초과
- `E404-PRESCRIPTION`: 처방 미존재
