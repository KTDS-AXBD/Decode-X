# AG-006 — 작물 인증 atomic

**BL ID**: AG-006
**Domain**: Agriculture (농업)
**Detector**: AtomicTransaction
**Function**: `issueCertification()`

## Condition
작물 인증 발급 시 (organic / gmo-free / fair-trade)

## Criteria
- crop.status = 'harvested' or 'sold'
- crop.grade != 'rejected'

## Outcome
atomic: certifications INSERT + certification_labels INSERT + crops.certified=1

## Exception
- `E404-CROP`: 작물 미존재
- `E409-CROP`: 인증 불가 상태 (미수확/미판매)
- `E422-CROP-REJECTED`: 등급 rejected 작물 인증 불가
