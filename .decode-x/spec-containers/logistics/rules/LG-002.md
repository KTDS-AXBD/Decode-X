# LG-002 — 경로 거리 한도 검증

**BL ID**: LG-002
**Domain**: Logistics (물류)
**Detector**: ThresholdCheck (Path B — var-vs-var, `maxRouteLimit` keyword 'limit' 매칭)
**Function**: `optimizeRoute()`

## Condition
화물 경로 최적화 요청 시 (shipment.status='pending')

## Criteria
- `routeDistanceKm ≤ maxRouteLimit` (20,000 km)

## Outcome
`shipments.route_distance_km` UPDATE, `withinLimit=true` 반환

## Exception
- `E404-SH`: 화물 미존재
- `E409-ST`: 상태 불일치 (pending 아님)
- `E422-LIMIT`: 경로 거리 초과
