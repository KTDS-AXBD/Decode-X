# DF-006 — 기밀 문서 처리 atomic

**함수**: `processClassifiedDocument()`
**Detector**: AtomicTransaction

## 조건
기밀 문서 처리 요청 시 분류 + 검증 + 발급 + 감사 원자적 처리.

## 기준
문서 처리 요청 필수 (classification_level: confidential | secret | top_secret)

## 결과
atomic: classified_documents INSERT + document_validations INSERT + document_issuances INSERT + document_audit_logs INSERT + classified_documents.status UPDATE

## 근거
기밀 문서 발급은 분류 + 검증 + 수령 기록 + 감사 로그가 원자적으로 완결되어야 함. 부분 실행은 허용되지 않는다.
