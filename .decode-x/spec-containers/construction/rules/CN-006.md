# CN-006 — 안전 검사 원자 트랜잭션

**BL ID**: CN-006
**Domain**: Construction (건설)
**Detector**: AtomicTransaction
**Function**: `processSafetyInspection()`

## Condition
안전 검사 처리 시

## Criteria
- `project.status = 'in_progress'`

## Outcome
atomic: `safety_inspections` INSERT + `correction_orders` INSERT(실패 시) + `projects.last_inspection_*` UPDATE

## Exception
- `E404-PROJECT`: 프로젝트 미존재
- `E409-PROJECT`: 비정상 프로젝트 상태 (in_progress 아님)
