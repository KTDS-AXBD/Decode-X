# Runbook — AG-003: 수확 처리 atomic

## 트리거
검사원/수확관리SYSTEM → `POST /api/v1/agriculture/crops/:id/harvest`

## 처리 흐름
1. crop.status = 'mature' 확인
2. field.status != 'restricted' 확인
3. 수확량 기준 등급 자동 산출 (A: ≥5000kg / B: ≥2000kg / C: 나머지)
4. atomic transaction: harvests INSERT + gradings INSERT + crops.status='harvested'

## 모니터링
- 수확 처리 성공률
- E422-FIELD-RESTRICTED 발생률 (제한 구역 수확 시도)
- 등급 분포 (A/B/C 비율)
