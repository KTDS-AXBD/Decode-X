# Runbook — RA-001 채널 동시 active program 한도

## 알림 조건
`E422-CHANNEL-CAPACITY-EXCEEDED` 빈도 급증 시.

## 진단
1. `channels.active_programs` vs `total_capacity` 비교 조회
2. 특정 채널/시간대(prime time) 동시 편성 집중 여부 확인
3. `MAX_CONCURRENT_ACTIVE_PROGRAMS_PER_CHANNEL` 기본값(48) 적절성 검토 (30분 슬롯 vs 1시간 슬롯)

## 해소
- 임시: 채널에 archived 후보 program 정리 권유 또는 total_capacity 직접 조정
- 영구: 시간대 분할 정책 명문화 + 추가 채널 (sub-channel) 신설
