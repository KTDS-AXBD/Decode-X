# TS-006 — 운행 중단 환불 atomic

**함수**: `processSuspensionRefund()`
**Detector**: AtomicTransaction

## 조건
`route.status = 'suspended'` 상태 노선에서 환불 요청.

## 기준
환불 금액 + 보상 금액 합산 원자적 처리.

## 결과
단일 트랜잭션: suspension_refunds INSERT + refund_records INSERT + routes UPDATE(suspension_refund_issued=1)

## 예외
`E409-ROUTE` — suspended 상태가 아닌 경우
