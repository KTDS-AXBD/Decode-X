# AG-004 — 작물 상태 전환

**BL ID**: AG-004
**Domain**: Agriculture (농업)
**Detector**: StatusTransition
**Function**: `transitionCropStatus()`

## Condition
작물 상태 전환 (planted → growing → mature → harvested → sold)

## Criteria
허용 매트릭스 충족:
- planted → growing
- growing → mature
- mature → harvested
- harvested → sold

## Outcome
crops.status UPDATE + 타임스탬프 기록 (harvested_at / sold_at)

## Exception
- `E404-CROP`: 작물 미존재
- `E409-CROP`: 허용되지 않은 상태 전환
