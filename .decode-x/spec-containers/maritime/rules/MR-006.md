# MR-006 — 손상 신고 atomic

**BL ID**: MR-006
**Domain**: Maritime (해운)
**Detector**: AtomicTransaction
**Function**: `processDamageClaim()`

## Condition
손상 신고 처리 시

## Criteria
- shipment.status = 'delivered' 또는 'arrived'

## Outcome
atomic: damage_claims INSERT + compensation_records INSERT + shipments.damage_claim_filed UPDATE

## Exception
- `E404-SHIPMENT`: 선적 미존재
- `E409-SHIPMENT`: 비정상 선적 상태 (delivered/arrived가 아닌 경우)
