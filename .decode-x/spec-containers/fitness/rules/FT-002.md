# FT-002 — 멤버십 PT 세션 한도 비교

**함수**: `usePtSession()`
**Detector**: ThresholdCheck (Path B, var-vs-var, ptSessionLimit keyword)

## 조건
PT 세션 사용 요청 시 `membership.pt_sessions_used >= ptSessionLimit` 검사.

## 기준
`ptSessionLimit = memberships.pt_session_limit` (멤버십 등급별 PT 세션 한도)

## 결과
- 한도 내: PT 세션 허용, pt_sessions_used 증가
- 한도 초과: `E422-PT-SESSION-LIMIT-EXCEEDED` 예외

## 근거
피트니스 멤버십 등급에 따른 월간 PT 세션 이용 한도 관리.
